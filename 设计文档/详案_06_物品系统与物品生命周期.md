# 详案_06：物品系统与物品生命周期 (GDD_06)

> **定位说明：** 本文档是全游戏**物品数据模型**的核心标准。背包战斗（GDD_01）、深渊探索（GDD_02）、模拟经营（GDD_04）三个系统的数据流动均以本文档中的定义为基础。程序在搭建物品数据结构时应以此为第一参考。

---

## 第一部分：核心设计哲学

### 1. 万物皆 Entity，功能由 Component 决定
本游戏彻底废弃"武器类 / 防具类 / 杂物类"的传统继承式分类。所有进入游戏流转的物体，无论是一把重型机枪还是一个占格子的毛绒玩具，在代码层面统称为 **`ItemEntity`（物品实体）**。  
物品"是什么"由挂载的组件（Component）决定，组件的组合方式由外部配置表（JSON5 / Lua）驱动，不写死在代码里。

### 2. 形态是最顶级的货币
在本游戏的经济模型中，1 个背包网格格子 = **战斗力 + 搜刮空间** 的双重价值体。物品占用的**形状（Shape）**即是它在空间上对玩家施加的"税"。一件物品是否值得带入、值得保留，核心成本单位不是金币，而是格子。

### 3. 物品的"双重身份"
每件物品同时具备：
- **静态定义（Item Definition）**：存在配置表中，描述这类物品的形状、标签、基础效果——这是模板，永远不变。
- **动态实例（Item Instance）**：游戏运行时生成，拥有唯一ID，记录该实例当前的磨损度和附加污染——同一类物品的不同实例可以有不同的"历史状态"。

## 第二部分：物品实体的组件模型

每个 `ItemEntity` 由以下**组件（Component）**按需组合：

### 2.1 形态组件（Shape Component）— 必选
定义物品在背包网格中占据的二维形状。

| 字段 | 类型 | 说明 |
|---|---|---|
| `GridPattern` | `int[][]` | 二维数组，1=占格，0=空，以左上角为原点 |
| `CanRotate` | `bool` | 是否允许玩家在局内旋转（R键） |
| `RotationSteps` | `int` | 旋转次数限制（1=只能横/竖，4=完整4方向） |

**形状优先级原则**：越高价值的物品设计越不规整的形状（L形、T形、凹形），用形状本身来制造放置难度和策略性。

```json
// 示例：L形宝箱（占5格，极难嵌入）
"GridPattern": [
  [1, 0],
  [1, 0],
  [1, 1]
]
```

---

### 2.2 标签组件（Tag Component）— 必选
物品的"身份证"，驱动所有系统间的联动判断。一件物品可同时拥有多个标签。

#### 战斗流派标签（影响背包 Synergy 计算）
| 标签 | 含义 |
|---|---|
| `[机械]` / `[Gear]` | 机械类武器/装置，被"超频"天赋增益 |
| `[血肉]` / `[Organic]` | 生物/肉质素材，被"生物质熔炉"消耗 |
| `[魔法]` / `[Magic]` | 魔法类武器，被法师义体加成 |
| `[斩击]` / `[Slash]` | 斩击型近战武器 |
| `[射击]` / `[Ranged]` | 枪械/投射类 |
| `[导体]` / `[Conductor]` | 连接器/管道，用于传输能量至相邻武器 |

#### 物品用途标签（影响经济流转）
| 标签 | 含义 |
|---|---|
| `[贸易品]` / `[Trade]` | 主要用于店铺出售换钱，无战斗意义 |
| `[素材]` / `[Material]` | 主要用于工坊制造/改造，出售价值低 |
| `[消耗品]` / `[Consumable]` | 使用后消失，使用时释放空间 |
| `[装备]` / `[Equipment]` | 持久使用的战斗物品，可带入下潜 |
| `[遗物]` / `[Relic]` | 深渊特有，局内被动效果，撤离后可研究 |
| `[情感锚点]` / `[Bond]` | 羁绊道具，无战斗力，具有情感机制 |
| `[委托物]` / `[Quest]` | 势力订单目标物，活着带出才算完成 |

#### 风险标签（影响污染/侵蚀逻辑）
| 标签 | 含义 |
|---|---|
| `[诅咒]` / `[Cursed]` | 向周围8格辐射瘴气，加速SAN流失 |
| `[高侵蚀]` / `[HighCorrupt]` | 每回合增加魔偶侵蚀度，黑市才肯高价收 |
| `[活物]` / `[Living]` | 物品本身会在背包中随机移动（如深渊寄生虫） |

---

### 2.3 触发组件（Trigger Component）— 可选
定义物品在战斗中如何"发作"。

#### 主动型（Active）
物品有充能条，满后自动或手动触发。
```json
{
  "TriggerType": "Active",
  "FillTime": 3.5,        // 充能时间（秒）
  "OnFire": {
    "ActionType": "DamageEnemy",
    "Value": 12,
    "DamageType": "Physical"
  }
}
```

#### 被动型（Passive）
监听特定事件，满足条件时自动触发。
```json
{
  "TriggerType": "Passive",
  "ListenEvent": "OnAdjacentItemFire",  // 监听：相邻物品开火时
  "Condition": { "RequireTag": ["Magic"] },
  "OnTrigger": {
    "ActionType": "AddBuff",
    "BuffID": "magic_chain",
    "Duration": 2.0
  }
}
```

#### 消耗型（Consumable）
需要玩家手动拖拽到目标上使用，使用后物品销毁并释放空间。
```json
{
  "TriggerType": "Consumable",
  "OnUse_Target": "Self",
  "OnUse": {
    "ActionType": "ModifyHP",
    "Value": 30
  }
}
```

---

### 2.4 空间干涉组件（Spatial Component）— 可选
物品对周围格子产生的影响，是"构筑链路流"的核心。

#### 正向干涉（连接/光环）
```json
{
  "AuraType": "Buff_FillSpeed",
  "TargetDirection": "LeftAndRight",  // 作用方向
  "TargetRange": 1,                   // 格子距离
  "RequireTag": ["Melee"],            // 目标需满足的标签
  "Value": 0.5,
  "IsStackable": false
}
```

#### 负向干涉（辐射/污染）
```json
{
  "AuraType": "Debuff_FillSpeed",
  "TargetDirection": "AllAdjacent8",
  "TargetRange": 1,
  "Value": -0.2,
  "SideEffect": "AddSAN_Drain",  // 同时给魔偶加侵蚀
  "SideValue": 1
}
```

---

### 2.5 实例状态组件（Instance State Component）— 运行时附加
仅存在于运行时的物品实例（Item Instance）上，不写入配置表。

| 字段 | 类型 | 默认值 | 说明 |
|---|---|---|---|
| `InstanceID` | `string` (UUID) | 生成时赋予 | 唯一标识符 |
| `Durability` | `float` (0~1) | 1.0 | 磨损度，0时物品效果大幅减弱 |
| `CorruptionLevel` | `int` (0~5) | 0 | 深渊污染等级，影响侵蚀辐射强度 |
| `DynamicTags` | `string[]` | `[]` | 运行时附加的临时标签（如被诅咒后加`[腐化]`） |
| `IsBinding` | `bool` | false | 是否已绑定（绑定后战斗失败不丢失） |

## 第三部分：物品的三大分类与设计逻辑

### 3.1 带入深渊的物品（Input Items）— 成本/生存端

> 这些物品从玩家仓库/工坊带入深渊。带的越多越安全，但初始搜刮空间越少。

#### A. 核心武装（长期占位）
- **定位**：构筑（Build）的框架，通常从局外带入，深渊全程占格。
- **设计原则**：越强的武装 → 形状越大/越奇葩 → 限制玩家的搜刮灵活性。
- **代表物品**：

| 物品名 | 形状 | 核心标签 | 特征 |
|---|---|---|---|
| 重型机导炮 | 3x2 / 凹形 | `[机械][射击][重型]` | 高伤害，需要能量核心相邻才能触发 |
| 灵能感应剑 | 1x3 / 纵向 | `[斩击][魔法]` | 传统链路流主力，C位武器 |
| 护盾发生器 | 2x2 / 方形 | `[机械][防具]` | 被动型，受攻击时自动消耗充能护盾 |
| 魔力核 | 1x1 / 点 | `[魔法][导体][能量源]` | 本身无伤害，向相邻格输出能量供消耗端使用 |

#### B. 战术消耗品（空间释放者）
- **定位**：有限的、用完后消失的格子，本质是"预载的空间"。深渊里使用消耗品 = 腾格子装战利品。
- **设计原则**：鼓励玩家"故意去找怪碰瓷把药喝掉"来获得背包空间。

| 物品名 | 形状 | 效果 | 设计痛点 |
|---|---|---|---|
| 修复药水 | 1x1 | 回复30 HP | 极小但极珍贵 |
| 护盾电池 | 1x2 | 恢复防具一层护盾 | 横放/竖放选择增加策略性 |
| 理智镇定剂 | 1x1 | 回复20 SAN | 深渊必备，高价值 |
| 深渊解毒剂 | 1x1 | 清除1层[诅咒]污染 | 在携带诅咒物品时必需 |

#### C. 功能性工具（投资型物品）
- **定位**：专门触发地图特殊事件的"钥匙"道具，赌博性质极强。
- **设计原则**：如果带了但没遇到对应节点，就是纯纯的"占位废物"。

| 物品名 | 形状 | 触发条件 | 效果 |
|---|---|---|---|
| 深渊矿镐 | 1x2 | 进入矿石节点 | 可开采稀有矿石（素材） |
| 万能钥匙 | 1x1 | 进入上锁宝箱节点 | 打开获得高品质战利品 |
| 诱饵发生器 | 2x1 | 主动使用 | 诱引精英怪，掉落率提升，高风险 |
| 深渊探测仪 | 1x1 | 被动，常驻 | 提前显示下一未知节点的类型 |

#### D. 情感锚点物（羁绊联动）
- **定位**：无任何战斗数值，纯粹通过好感度系统和深渊情感机制产生价值。
- **设计原则**：这是将"好感度系统"与"背包核心玩法"物理绑定的关键道具——玩家必须用格子换情感。

| 物品名 | 形状 | 情感机制 |
|---|---|---|
| 旧怀表 | 1x1 | 在包中时，每层侵蚀度增长-10%（随好感度等级提升效果） |
| 小熊玩偶 | 2x2 | 保护周围格子免受深渊结晶封锁；满好感时，致死触发锁血 |
| 八音盒 | 2x2 | 高压战斗中为相邻武器附魔[守护]属性；满好感时，周围武器临时升级 |
| 兔子挂饰 | 1x1 | 带在包中撤离时，额外获得一次"无损撤离"判定（不计入该次战斗的SAN损耗） |

### 3.2 深渊产出的物品（Output Items）— 收益/目标端

> 这些物品在深渊中通过战斗、宝箱、事件获得。它们是驱动玩家回城的核心动力。

#### A. 纯贸易品（死板的财富）
- **定位**：唯一意义是带回小镇卖钱，无任何战斗价值。
- **设计原则**：故意设计成极不规整的形状（3x3方块、L形），制造"为了钱扔武器"的核心矛盾。

| 物品名 | 形状 | 基础价值 | 特殊说明 |
|---|---|---|---|
| 古代金币（整袋） | 2x1 | 300G | 形状规整，常见，低端 |
| 深渊珍珠 | 1x1 | 500G | 极小，高价，但手感容易"不可见"（掉落不显眼） |
| 名贵画作 | 2x3 / 纵向 | 1500G | 巨占格子，典型的"贪婪诱惑物" |
| 古代雕像 | 3x3 / 方块 | 3000G | 顶级诱惑物，装下它基本等于自杀 |
| 魔法水晶球 | 2x2 | 800G | 中规中矩，偶有奇特的隐藏鉴定价值 |

#### B. 素材与异化器官（养成材料）
- **定位**：出售价值低，但工坊制造时不可缺少。玩家需要明确的下潜目的："这次只为了打3个晶化心脏"。

| 物品名 | 形状 | 工坊用途 |
|---|---|---|
| 深渊蜘蛛腿 | 1x3 / 纵向 | 制造[影袭底盘]配件、灵活型义体 |
| 晶化心脏 | 1x1 | 稀有，用于解锁最高阶底盘升级 |
| 机械残骸 | 2x1 | 普通义体插件的基础材料 |
| 净化结晶 | 1x1 | 机体维护中消耗，保养核心材料 |
| 深渊熔融矿石 | 2x2 | 高阶武器锻造原料，低售价 |

#### C. 深渊遗物（局内肉鸽增益）
- **定位**：类《杀戮尖塔》遗物。捡到后塞入背包提供全局被动效果，撤离后可研究转化。
- **设计原则**：遗物必须占据至少1格，不能是"无形的"被动奖励——玩家必须为它付出空间代价。

| 遗物名 | 形状 | 局内效果 | 撤离后研究效果 |
|---|---|---|---|
| 深渊侵者之眼 | 1x1 | 每击杀一个怪物，自身同标签武器冷却-5% | 转化为义体插件："增加深渊地图隐藏节点显示率" |
| 腐化铁王冠 | 2x2 | `[诅咒]`物品的辐射范围缩减为0，改为提升自身全武器攻击力 | 解锁"异化冶炼图纸"制造稀有武器 |
| 共鸣晶簇 | 1x2 | 背包内同标签物品数量 × 2% 全局暴击率 | 转化为"同调材料"供底盘升级消耗 |
| 虚空锚点 | 1x1 | 在安全区可额外打包一格物品进"锚定收纳盒"（不计入有限容量） | 研究失败，但获得大量金币补偿 |

#### D. 势力订单物品（委托目标）
- **定位**：小镇势力发布的专属收购令，必须活着带出深渊才能交差。
- **设计原则**：故意让这些物品占据大量背包空间，形成"局内的强制生存压力"。

| 委托物名 | 形状 | 对应势力 | 委托奖励 |
|---|---|---|---|
| 深渊熔炉核心 | 3x3 / 方块 | 机械工坊 | 大量金币 + 底盘图纸 |
| 活捕深渊史莱姆（捕笼） | 2x2 | 法师塔 | 解锁传送卷轴图纸 |
| 深渊龙血（密封瓶） | 2x2 | 炼金公会 | 高额金币，急需时翻3倍物价 |
| 古代石碑拓本 | 1x3 / 横向 | 冒险者公会 | 声望+提高隐藏地图发现概率 |

### 3.3 两栖物品（Pivot Items）— 策略爆发点

> 模糊"带入"与"产出"界限的物品。它们是游戏中最好玩的部分——玩家需要在局内临时决定它的用途。

#### A. 深渊临时武装（捡垃圾流）
从怪物掉落，可在局内临时塞进构筑使用，撤离后当素材/废铁卖掉。

| 物品名 | 形状 | 战斗用途 | 经济用途 |
|---|---|---|---|
| 生锈的毒刃 | 1x2 | 临时附毒快攻武器 | 卖50G废铁价 |
| 哥布林骨弓 | 2x1 | 低伤远程武器，打远怪 | 卖30G |
| 深渊污液罐（手雷型） | 1x1 | 消耗型，投掷造成毒AOE | 卖100G（当素材用） |

#### B. 两用型战利品（终极抉择）
玩家必须当场决定：战斗用掉损失经济价值，经济变现放弃战斗力。

| 物品名 | 形状 | 战斗用途 | 经济价值 | 玩家的纠结 |
|---|---|---|---|---|
| 深渊爆裂腺体 | 2x1 | 当场扔出，全屏毁灭性伤害 | 5000G | 保命 vs 5000G |
| 深渊放射核（封装） | 1x1 | 高额被动伤害Buff，使相邻武器攻击力+100% | 3000G | 强力增益 vs 3000G |
| 精油蚀线源 | 2x2 | 主动使用：使所有怪物中毒10秒 | 1500G | 困难时的救命稻草 |

#### C. 被污染的极品装备（风险利用）
深渊深处出现，面板极高但附有强烈负面词条，玩家需要在利用价值和风险之间取舍。

| 物品名 | 形状 | 正面效果 | 负面词条 | 利用思路 |
|---|---|---|---|---|
| 嗜血魔剑 | 1x3 | 攻击力翻倍 | 每回合+5侵蚀度 | 快速冲刺清图后赶紧撤离 |
| 寄生裂甲 | 2x2 | 护盾再生翻倍 | 每受一次攻击，随机封锁1个空格 | 必须确保快速击杀，不能挨打 |
| 诅咒圣遗器 | 1x1 | 局内所有战利品掉落率+50% | `[诅咒]`辐射全背包 | 塞进绝缘铅盒（带入道具）使用 |

## 第四部分：物品生命周期漏斗模型

```
[ 深渊中 ]         [ 分流点 ]          [ 地面消化 ]
  怪物/宝箱
  事件掉落  ──▶  临时战利品槽  ──▶  玩家塞入背包
                  (Inbox暂存)    │
                               未装入 ──▶  直接丢弃（永久消失）
                               │
                          ┌────┘
                          ▼
                    [ 背包持有中 ]
                    战利品 / 工具 / 武器共存
                          │
                     在安全区选择
                    ┌─────┴─────┐
                  撤离           继续深潜
                  │              │（下一层仍在背包）
                  ▼
            地面仓库（Stash）
                  │
          ┌───────┼───────┐
          ▼       ▼       ▼
       [贸易]  [素材]  [装备]
         │       │       │
       商店   工坊制造  下次下潜
       货架   消耗改造   带入
         │
       卖出→金币
```

### 关键规则
- **临时战利品槽（Inbox）**：战斗结束后，掉落物在Inbox停留至玩家离开当前节点为止。离开时Inbox未装入背包的物品**永久销毁**。
- **安全收纳盒（Safe Box）**：在安全区可将部分物品打包进"安全收纳盒"（容量有限，初始2格）。即使接下来战败，收纳盒内物品**不会丢失**。
- **战败丢失规则**：在非安全区战败，背包中所有**未绑定（IsBinding=false）**的物品全部丢失。**绑定物品**（情感锚点物自动绑定）和安全收纳盒中的物品保留。

## 第五部分：物品价值系统与定价框架

### 5.1 价值的三个维度
每件物品的"价值"不能只用金币衡量，必须从三个维度综合评估：

| 维度 | 含义 | 影响什么 |
|---|---|---|
| **经济价值（E）** | 在小镇商店的基础售价 | 决定卖出收益 |
| **战斗价值（C）** | 在背包中提供的DPS/生存贡献 | 决定是否值得占用格子 |
| **素材价值（M）** | 在工坊中作为制造原料的价值 | 决定是否值得存仓库 |

**设计黄金律**：物品的三个维度**不能同时拉满**。高E必然低C（纯贸易品）、高C通常低E（战斗装备）、高M通常形状奇特（制造素材）。

### 5.2 基础价格区间参考（以金币G为单位）

| 物品等级 | 经济价值（E）范围 | 代表物品 |
|---|---|---|
| 白（普通） | 10 ~ 100G | 史莱姆凝胶、哥布林牙齿 |
| 绿（稀少） | 100 ~ 500G | 深渊蜘蛛腿、净化结晶 |
| 蓝（稀有） | 500 ~ 2000G | 晶化心脏、深渊珍珠 |
| 紫（珍稀） | 2000 ~ 5000G | 深渊核心部件、遗物 |
| 橙（传说） | 5000G以上 | 深渊龙血、古代神器 |

### 5.3 动态物价调节
详见 **GDD_04**（小镇循环与经济物价波浪模型）。  
简要规则：物品基础价格 × `当前市场系数（0.3 ~ 3.0）` = 实际售价。  
市场系数由小镇传闻事件每周刷新驱动，玩家可通过情报板提前获知。

## 第七部分：边界情况处理（Edge Cases）

| 极端情况 | 处理规范 |
|---|---|
| `[活物]`标签物品（如深渊寄生虫）移动到了已被深渊结晶封锁的格子上 | 活物在移动前检测目标格合法性，若为`-1`封锁格则跳过此次移动，停留原位，并触发自身受1点伤害（产生疲惫效果） |
| 玩家将`[情感锚点]`物品放入安全收纳盒，而非带在背包里 | 情感锚点物品的被动效果（SAN减缓、锁血机制）**仅在背包网格中生效**，放入收纳盒不触发任何情感机制 |
| 两件具有负向干涉的`[诅咒]`物品的辐射范围重叠 | 不叠加，取各物品各自的最高值作用于目标格，而不是相加（防止多个诅咒物叠加导致数值崩盘） |
| 消耗型物品在战斗中因怪物技能被"销毁"（非玩家主动使用） | 触发物品的`OnDestroy`事件。此事件不等同于`OnUse`，不触发使用效果和腾出空间奖励，仅释放格子空间 |
| 携带委托物（`[委托物]`标签）在战败后被判定为丢失 | 委托物和普通战利品一样遵循"未绑定则丢失"规则。委托失败需玩家重新下潜再次获取，势力订单的截止期限继续倒计时 |
| 鉴定后发现"未鉴定遗物"是已经拥有的同款遗物 | 若玩家背包/仓库中已存在同`DefinitionID`的遗物实例，系统提示"已持有同款"，但允许继续保留（可拿去研究/出售，不强制销毁） |
