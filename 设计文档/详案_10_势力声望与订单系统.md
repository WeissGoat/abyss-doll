# 详案_10：势力声望与订单系统 (GDD_10)

> **定位说明：** 势力系统是小镇社会的"骨架"。每个势力代表一种资源流向和游戏风格的偏好，玩家通过与不同势力打交道获取独占资源（图纸、商品、情报），但时间和精力有限，不可能同时讨好所有人。订单系统则是势力施加在玩家身上的"定向任务压力"，以极端的背包空间代价换取丰厚的定向回报。
>
> **与其他文档的关系：**
> - 声望奖励中的图纸 → 解锁 GDD_08（养成系统）中的底盘和义体
> - 订单的委托物 → 占据 GDD_01（背包战斗）的核心空间
> - 订单截止日 → 绑定 GDD_07（时间日程）的周结算
> - 势力商店 → 扩展 GDD_04（经济系统）的商品渠道

---

## 一、势力的设计哲学

### 1.1 势力不是阵营——是资源管道
本游戏的势力不涉及"善恶对立"或"阵营站队"。每个势力都是中性的利益团体，它们和玩家之间的关系是纯粹的**交易关系**：你替我做事，我给你好处。

玩家不需要"加入"任何势力。声望只代表合作深度——做的越多，对方越信任你，愿意拿出更好的东西来交换。

### 1.2 为什么需要势力系统？
在没有势力系统的情况下，玩家的行为模式是：下深渊 → 带回战利品 → 全卖掉 → 拿钱交租/买东西。

势力系统引入了**定向目标**：
- "法师塔这周要深渊龙血，完成了能解锁传送卷轴图纸"
- "机械工坊的委托：带回一个3×3的熔炉核心，报酬是底盘蓝图"

这把"漫无目的的搜刮"变成了"有明确目标的猎取任务"，极大提升了每次下潜的方向感。

## 二、势力图鉴

小镇中共有 **4个主要势力** 和 **1个隐藏势力**。

### 2.1 冒险者公会

| 属性 | 内容 |
|---|---|
| **定位** | 深渊探索者的组织，最容易打交道的势力 |
| **世界观** | 一群退役冒险者组成的互助会，负责管理深渊入口和登记深潜者 |
| **核心提供** | 深渊情报（地图信息）、基础消耗品批发、安全收纳盒扩容服务 |
| **偏好的交易品** | 深渊地图拓本、怪物图鉴标本、探索记录 |
| **性格** | 务实、豪爽、不问出身 |

**声望奖励阶梯：**

| 声望等级 | 所需累计贡献 | 解锁奖励 |
|---|---|---|
| Lv.1 认识 | 完成首次委托 | 解锁公会商店（基础消耗品8折） |
| Lv.2 熟人 | 累计完成3次委托 | 安全收纳盒扩容+1格 |
| Lv.3 伙伴 | 累计完成7次委托 | 解锁「深渊探测仪」图纸（进深渊前预览下一层主题） |
| Lv.4 英雄 | 累计完成12次委托 + 到达深渊7层 | 深渊入口处可购买"紧急撤离信号弹"（在非安全区强制撤离，代价：丢失50%战利品） |
| Lv.5 传奇 | 累计完成20次委托 + 到达深渊9层 | 解锁隐藏的深渊第10层入口 + 「影袭底盘」蓝图 |

| Lv.5 塔主候选 | 解锁深渊9层以下的特殊遗物并全数交付 | 解锁隐藏魔法武器锻造 + 心智天赋树「第四隐藏分支」入口 |

| 属性 | 内容 |
|---|---|

| **世界观** | 控制着小镇大部分药剂流通的商人联合会。表面温文尔雅，实则精于算计 |
| **核心提供** | 高级消耗品配方、药剂批发折扣、物价内幕情报 |
| **偏好的交易品** | 深渊生物素材（蜘蛛腿、龙血等有机物）、稀有草药 |
| **性格** | 圆滑、重利、信守契约 |

**声望奖励阶梯：**

| 声望等级 | 所需累计贡献 | 解锁奖励 |
|---|---|---|

---

### 2.5 黑市（隐藏势力）

| 属性 | 内容 |
|---|---|
| **定位** | 灰色地带。危险品和违禁品的唯一流通渠道 |
| **世界观** | 没有固定据点。一个戴面具的商人在随机夜晚出现在工坊后巷 |
| **核心提供** | 诅咒物品的高价收购、被污染的极品装备、违禁义体（有副作用的高性能配件） |
| **偏好的交易品** | `[诅咒]`标签物品、`[高侵蚀]`标签物品、人偶部件（道德灰色） |
| **性格** | 神秘、不问来源、不讲价 |

**声望机制（特殊）：**
黑市没有传统的声望等级，而是用「信任度」衡量——每次成功交易+1信任度，爽约或举报-3信任度。

| 信任度 | 解锁内容 |
|---|---|
| 3 | 商人开始出售「违禁义体插件」（效果强但附带永久副作用） |
| 7 | 可以向商人出售势力委托物（背叛正规势力换取黑市独有材料） |
| 12 | 解锁「深渊走私通道」——绕过Boss直接进入深层（但不获得Boss掉落物） |
| 15 | 商人透露自己的真实身份，解锁隐藏剧情线 |

**风险**：如果其他正规势力发现玩家与黑市交易频繁（信任度超过10），正规势力的态度会变冷——进货价上涨、委托优先级降低。

## 三、订单（委托）系统

### 3.1 订单是什么
订单是势力发布的**定向采购需求**。玩家接单后需要在截止日前从深渊中获取指定物品并交付。

订单不是"支线任务"——它是用**背包空间和深渊风险**换取**声望和独占奖励**的核心交易手段。

### 3.2 订单的类型

#### A. 采购订单（最常见）
- **内容**：交付指定数量的指定物品
- **例子**：「机械工坊需要3块深渊矿石，截止日：本周末」
- **特点**：物品可以分批交付，不一定一次下潜搞定

#### B. 活体订单（高难度）
- **内容**：使用特殊容器在深渊中捕获活体目标并带回
- **例子**：「法师塔需要1只活捕深渊史莱姆，需使用捕笼（占2×2格）」
- **特点**：捕笼在局前就要带入背包，战斗全程占格子，且捕获后格子继续被占——极端的空间压力

#### C. 收购指定品（中等难度）
- **内容**：在特定层级击杀特定怪物/打开特定宝箱后获得任务物品
- **例子**：「炼金公会需要深渊龙血（密封瓶，占2×2），来源：第5层Boss」
- **特点**：物品的获取有概率性或关卡性，可能需要多次下潜尝试

#### D. 探索报告订单（低难度）
- **内容**：到达指定深渊层数并撤离
- **例子**：「冒险者公会需要深渊第6层的环境报告，只需活着回来即可」
- **特点**：不需要带回物品，但必须实际到达指定层数并安全撤离

### 3.3 订单的生成与刷新
- 每**周结算**时，各势力的订单池刷新
- 同时可见的待接订单数量有限（每个势力同时提供1~2个）
- 玩家可以同时接多个势力的订单，但背包空间是共享的
- **同时接了2个需要大型委托物的订单** = 背包几乎没有战斗空间 = 极限挑战

### 3.4 订单的截止机制

| 截止情况 | 后果 |
|---|---|
| **按时完成** | 获得声望 + 金币/物资奖励 + 势力好感 |
| **超期未完成** | 该订单作废，扣除少量声望，该势力短期内不再发布新委托 |
| **主动放弃** | 随时可以在订单面板取消已接受的订单。不扣声望，但占用了该势力的委托槽位直到下次刷新 |

### 3.5 订单对背包的压力——核心体验

这是订单系统存在的真正意义。

**场景示例**：
> 你接了机械工坊的委托——带回一个「深渊熔炉核心」（3×3的巨型方块）。  
> 你的底盘是5×5的标准方格。  
> 3×3的委托物一旦找到并塞入背包，它将占据背包面积的36%。  
> 这意味着这次下潜你的战斗构筑只剩64%的空间来摆放武器和防具。  
> 你必须用极其精简的武装挺过Boss战，才有资格拿到核心。  
> 然后你必须带着这个塞满了大半个背包的笨重物体，在仅剩的空间里靠仅存的武器撑到安全区撤离。

这种"带着巨大累赘物品在深渊中求生"的体验，是本游戏独有的、无法被其他游戏复制的设计——因为它依赖"背包即战场"这个核心前提。

## 四、势力之间的博弈

### 4.1 时间竞争
玩家的AP（行动点）是恒定的。拜访势力交付物品、接取委托、使用势力服务——都需要消耗日程中的AP。你不可能一天之内跑遍所有势力。

### 4.2 资源竞争
深渊的战利品是有限的。同一块深渊矿石，你交给机械工坊能换义体图纸，交给炼金公会能换药剂配方，卖给黑市能换违禁物品。**但你只有一块矿石。**

### 4.3 隐性对立
部分势力之间存在暗流：
- **法师塔 vs 机械工坊**：两者都想要深渊Boss核心。玩家交给谁，另一方心里有数
- **炼金公会 vs 黑市**：炼金公会痛恨黑市的违禁品交易。如果玩家同时拥有两者的高声望，炼金公会会在对话中旁敲侧击
- **冒险者公会**：唯一的中立势力。和任何人交好都不影响公会关系

### 4.4 不存在"全部满声望"
声望总量经过设计控制——一局游戏（12个月）中，受制于时间和素材稀缺性，玩家最多只能将 **2~3个势力**推到高声望（Lv.4~5），其余势力只能维持在中低水平。

这迫使玩家根据自己的构筑流派和养成方向做选择：
- 走重火力流 → 优先机械工坊（底盘+义体）
- 走深渊探索流 → 优先冒险者公会（安全盒+情报）
- 走经营流 → 优先炼金公会（药剂+物价情报）
- 走神秘学流 → 优先法师塔（遗物+心智树）

## 五、势力NPC与叙事功能

### 5.1 每个势力由一个核心NPC代表

| 势力 | NPC名称 | 性格关键词 | 与玩家的关系弧 |
|---|---|---|---|
| 冒险者公会 | 公会老板娘 | 豪爽、照顾后辈、偶尔八卦 | 从"看不起新人"到"把你当自家孩子" |
| 机械工坊 | 偏执老工匠 | 沉默、只看技术说话 | 从"嫌你浪费材料"到"承认你是天才" |
| 法师塔 | 年轻女法师 | 傲娇、对深渊着迷 | 从"你不过是个搬运工"到"你是我唯一的合作伙伴" |
| 炼金公会 | 微笑会长 | 文雅、腹黑、守信 | 从"客客气气的交易"到"利益共同体的默契" |
| 黑市 | 面具商人 | 不可读、暧昧 | 从"匿名买卖"到"揭开面具的真相" |

### 5.2 势力对话的触发机制
势力NPC的对话不是线性剧本。它们由**声望等级 + 全局旗帜**共同驱动（详见GDD_05）：

- 声望升级时 → 触发里程碑对话
- 按时完成高难度委托时 → 触发认可类对话
- 连续爽约或交付劣质品时 → 触发失望类对话
- 特定剧情旗帜触发时 → 解锁势力专属支线

### 5.3 势力与结局的关联
声望最高的势力会影响终局时"小镇的态度"：
- 冒险者公会声望最高 → 结局时公会成员为你站台
- 机械工坊声望最高 → 结局时获得独占的机械系终极装备
- 法师塔声望最高 → 结局时解锁深渊真相的额外线索
- 黑市信任度最高 → 解锁完全独立的隐藏结局分支

## 六、势力商店与独占商品

### 6.1 势力商店
每个势力在达到一定声望后开放专属商店。商店商品是该势力**独占的**——在小镇普通商店买不到。

| 势力 | 商店特色 | 独占商品举例 |
|---|---|---|
| 冒险者公会 | 基础消耗品批发（量大优惠） | 紧急撤离信号弹、深渊探测仪、加固安全盒 |
| 机械工坊 | 武器改造、义体制造原料 | 精密零件、高纯合金、底盘适配工具 |
| 法师塔 | 遗物相关服务和魔法素材 | 记忆碎片催化剂、遗物净化药水、封印石 |
| 炼金公会 | 高级消耗品和经营辅助 | 强效理智镇定剂、客流吸引香薰、情报密信 |
| 黑市 | 违禁品（高性能但有副作用） | 超载能量核（耐力+50%但SAN每秒-1）、深渊寄生虫武器 |

### 6.2 商品解锁与声望绑定
- 低声望时商店只有基础库存
- 每升一级声望解锁新商品
- 部分红色标记商品需要"完成特定委托"后才能购买（不只是声望够就行）

## 七、边界规则

| 情况 | 规则 |
|---|---|
| 同时接了两个势力的委托，都需要同一种素材怎么办？ | 素材消耗是独立的——交给A就不能再交给B。玩家需要下两趟深渊或在接单前就做好取舍 |
| 把委托物卖给了黑市而不是交给委托方？ | 委托自动判定为失败（超期未交付），扣除对应势力声望。黑市信任度正常增加。这是一种合法但有代价的"背叛"策略 |
| 势力声望能降到0以下吗？ | 不能。最低止于Lv.0。但连续多次爽约后，该势力会暂停向你发布委托（冷却期1~2周） |
| 多周目时声望继承吗？ | 不继承。每局重新建立关系。但已解锁的图纸配方在新周目的"已知配方库"中保留——你知道怎么做，但需要重新拿到材料和声望来执行 |
