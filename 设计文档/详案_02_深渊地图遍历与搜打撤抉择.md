# 详案_02：深渊地图遍历与搜打撤抉择 (GDD_02)

> **定位说明：** 本文档定义了深渊探索的完整流程——从下潜前的整备，到节点树地图的推进规则，到安全区的撤离博弈，再到失败后的惩罚机制。这是"搜打撤"三字的完整游戏体验设计。

---

## 一、深渊探索的整体流程

一次完整的深渊探索（下潜）分为四个阶段：

```
【整装待发】→ 【迷宫探索】→ 【安全区抉择】→ 【撤离结算 / 失败惩罚】
     ↑                                              │
     └──────────────────────────────────────────────┘
                    （返回地面，进入日常循环）
```

---

## 二、阶段一：整装待发（Pre-Dive）

### 2.1 仪式感与最后的策略调整
在玩家确认日程"下深渊"后、正式点击"下潜"按钮之前，进入「整备界面」。

**界面构成**：
- 左侧：魔偶的全身展示（显示当前底盘、义体、穿着的装甲）
- 右侧：背包网格 + 仓库面板，玩家在此完成最终的局前背包管理

### 2.2 局前背包管理的核心博弈
玩家需要从仓库中选择带入深渊的物品，塞入背包网格。

**核心矛盾**：
- 带的武器/防具越多 → 战斗越安全 → 但初始留给搜刮的空余格子越少
- 带的消耗品越多 → 续航越强 → 但使用前也在占格子
- 带的情感锚点物 → 保护心智 → 但没有任何战斗力

**预期玩家心理**：
>"我是塞满武器然后只捡小东西回来？还是轻装上阵只带一把匕首和三瓶药，赌一波大宝箱？"

### 2.3 委托物预留
如果玩家接了势力委托（如"带回一个3×3的深渊熔炉核心"），就必须在局前背包中**预留出相应大小的空位**来装委托物。这极大压缩了可用战斗空间，让"跑委托"成为高风险高回报的专项挑战。

### 2.4 魔偶状态检查
整备界面会醒目展示魔偶当前的HP/SAN/磨损度状态。如果状态不佳（参见GDD_07中的"状态对日程的反向约束"），界面会弹出警告提示。玩家可以选择无视警告强行下潜，但后果自负。

### 2.5 下潜前的情感互动
整备界面可以触发一次简短的魔偶反馈：
- 状态良好时："准备好了，主上，我们走吧。"（元气语音）
- 状态不佳时："主上……上次的伤还没好……但如果你需要的话……"（犹豫语音）
- 好感度极高时："不管前面是什么，有主上在我就不怕。"（温柔语音）

## 三、阶段二：迷宫探索（Exploration）

### 3.1 地图表现形式
进入深渊后，画面切换为2D的「节点树状图」（参考《杀戮尖塔》）。

- 地图由**节点和连线**组成，从下往上延伸
- 玩家通过**点选节点**前进，不是自由移动
- **不可回头**：走过的节点灰化，无法返回
- **战争迷雾**：只能看清当前节点和前方1~2层的分支，更远处被瘴气遮蔽

### 3.2 层级结构
深渊由多个"层（Layer）"纵向叠加，每一层是一张独立的节点网络。

| 概念 | 说明 |
|---|---|
| 层（Layer） | 深渊的一个大区段，有独立的主题和敌人池 |
| 层内节点 | 每层包含6~12个节点，通过分支路线连接 |
| 层末Boss节点 | 每层最后一个节点固定为Boss战 |
| 层间安全区 | 每层结束后（Boss之后）必定出现一个安全区 |

**层级推进**：
- 击败层末Boss → 进入层间安全区 → 选择撤离或深入下一层
- 总共设计约 **8~10层**，但一般玩家单次下潜只能推进2~4层

### 3.3 节点类型图鉴

#### 🗡️ 普通战斗节点（Minor Combat）
- 出现频率：最高（约50%）
- 内容：1~3只杂兵怪物
- 战利品：低价值小型物品（1×1或1×2），偶尔出现中等素材
- 体验：快节奏清理，消耗少量HP，积累零散战利品

#### ⚔️ 精英战斗节点（Elite Combat）
- 出现频率：中等（约15%）
- 内容：1只精英怪（或1只精英+1~2只杂兵）
- 战利品：必掉中高价值物品，概率掉落遗物
- 体验：真正的策略考验，需要消耗品支撑。是"高回报但高风险"的诱惑
- **地图标识**：精英节点在地图上用特殊颜色标识，玩家可以选择绕路避开

#### 💀 Boss战斗节点（Boss）
- 出现频率：每层末尾固定1个
- 内容：层Boss，阶段切换型战斗
- 战利品：必掉稀有/珍稀级物品，形状通常极大
- 体验：决定性关卡，是继续深入的门槛

#### ❓ 未知事件节点（Mystery/Event）
- 出现频率：中等（约15%）
- 内容：随机事件。可能是好事也可能是坏事
- 事件类型：
  - **祝福型**：发现遗迹宝箱（免费获得物品）
  - **交易型**：遇到流浪商人，可以用金币/物品换稀有道具
  - **献祭型**：祭坛要求牺牲HP/SAN/物品，换取强力临时增益
  - **陷阱型**：触发陷阱，背包中随机格子被封锁（SAN不足时陷阱概率大幅增加）
- **设计意图**：不确定性制造冒险感。高SAN时好事概率高，低SAN时坏事概率高——鼓励维护心智状态

#### 🎁 宝箱节点（Treasure）
- 出现频率：较低（约10%）
- 内容：免费获得1~2件战利品，无战斗
- 特殊规则：部分宝箱上锁（需要大地图上使用「万能钥匙」）；部分宝箱有诅咒陷阱
- 体验：纯粹的搜刮奖励，但可能引诱玩家为了装下宝箱物品而扔掉武器

#### 🏕️ 层内营地节点（Rest Stop）
- 出现频率：低（每层0~1个，出现在中段）
- 内容：不同于层间安全区，这是一个简易休息点
- 可用功能：
  - 恢复少量HP（消耗带来的修复材料）
  - 恢复少量SAN
  - **不能撤离**——只有层间安全区才能撤离
- 体验：喘息的机会，但不是逃生口

### 3.4 路线选择的策略性
每层地图中，玩家面临多条通往层末Boss的路线：

- **安全路线**：杂兵多、精英少，战利品平庸但风险低
- **冒险路线**：精英多、宝箱多，高风险高回报
- **探索路线**：未知事件密集，充满不确定性

路线的选择受当前状态强烈影响：
- 满血满SAN → 可以走冒险路线
- 残血低SAN → 被迫走安全路线
- 带了万能钥匙 → 优先走有宝箱的路线

### 3.5 环境侵蚀（过路税）
每移动一个节点，魔偶自动扣除少量SAN值——这是深渊的"过路税"。

- 基础扣除：每节点2点SAN
- 加重条件：
  - HP低于30%时，过路SAN消耗翻倍（机体暴露于瘴气，更脆弱）
  - 背包中携带`[高侵蚀]`标签物品时，过路SAN消耗+50%
  - 深层（Layer 5+）环境瘴气加重，基础消耗递增

**设计意图**：SAN是"计时器"。走的节点越多、SAN流失越快。这天然限制了玩家在一次下潜中能走多远，迫使适时撤离。

## 四、阶段三：安全区抉择（Safe Zone / Extraction Point）

### 4.1 安全区的位置
- **层间安全区**：每层Boss击败后必定出现，是唯一可以撤离的地点
- 层内营地不能撤离，只能短暂休整

### 4.2 安全区的视觉与氛围
- 画面从阴暗紧张切换为温暖安静：一团营火、一座古代传送门遗迹
- 魔偶放下背包，坐在营火旁。战损立绘展示（如果受伤严重的话）
- BGM从战斗紧张变为舒缓的环境音

### 4.3 安全区可用操作

#### A. 安全收纳盒打包
- 将背包中的部分高价值物品放入「安全收纳盒」
- 收纳盒中的物品**不算在背包构筑中**（无战斗效果），但**绝对安全**（即使之后战斗失败也不会丢失）
- 初始收纳盒大小仅2格，可通过局外投资扩大
- **设计意图**：让谨慎的玩家有"落袋为安"的安全感。但放进去就不能在后续战斗中使用了

#### B. 休整与维修
- 使用携带的修复素材恢复少量HP和SAN
- 消耗金币进行简易的机体修复（降低磨损度）
- 这些操作消耗的都是带入深渊的有限资源

#### C. 远程通讯（情感交互点）
- 玩家通过"通讯器"与魔偶进行简短对话
- 对话内容根据当前状态变化（满载而归时的喜悦 / 残血时的恐惧 / 好感度高时的信任）
- 此处的对话选择可能小幅影响好感度和SAN

#### D. 核心抉择：撤离还是深入

界面中央呈现两个醒目的选项：

**【撤离（Extraction）】**
- 确认后，魔偶启动传送门，画面切回工坊
- 背包中所有物品 + 安全收纳盒中物品全部安全转入地面仓库
- 结算本次深渊的总收益（战利品/素材/记忆碎片）
- 玩家感受："安全感" + "松了一口气" + 可能的"贪心未满足的遗憾"

**【深入下一层（Descend）】**
- 地图生成下一层更难、奖励更丰厚的节点网络
- 播放侵蚀度增加的动态效果
- 魔偶可能发出语音："主上，瘴气越来越浓了，我们真的要继续吗？"
- 玩家感受："贪婪" + "赌一把" + "如果死了之前的全白给"的恐惧

### 4.4 撤离时机的博弈
这是搜打撤玩法的灵魂——"到底什么时候该跑"：

| 当前状态 | 理性选择 | 贪婪的诱惑 |
|---|---|---|
| HP充足、SAN健康、背包小半满 | 可以继续 | "下面肯定有更好的东西" |
| HP充足但SAN偏低 | 应该撤 | "但下一层的Boss掉落能抵一个月房租" |
| 残血但背包满载 | 必须撤 | "可是还有两格空间不塞满太浪费了" |
| 满状态但背包已满 | 该撤了 | "但如果我扔掉那把剑就能再装一个遗物……" |

## 五、阶段四：结算——成功撤离 vs 失败惩罚

### 5.1 成功撤离结算
- 背包中所有物品（含绑定装备、战利品、两栖物品）安全转入地面仓库
- 安全收纳盒中物品安全转入地面仓库
- 本次下潜积累的记忆碎片归入心智天赋树可用总量
- 魔偶的HP/SAN/磨损度/侵蚀度状态带回地面，等待后续日程中安排修复

### 5.2 失败（战斗失败/SAN崩溃）惩罚

#### 物品丢失规则
| 物品类型 | 战败后处置 |
|---|---|
| 本次深渊新获得的战利品（`IsBinding=false`） | **全部丢失** |
| 安全收纳盒中的物品 | **保留** |
| 局前带入的绑定装备（`IsBinding=true`） | **保留**，但附加严重的「深渊损坏」标签（磨损度暴增） |
| 情感锚点物（`[Bond]`标签） | **保留**（自动绑定，不会因战败丢失） |
| 记忆碎片 | **保留**（非实体，不受战败影响） |

#### 魔偶状态惩罚
- HP清零（回到地面后从维修台醒来）
- SAN值大幅下降
- 磨损度暴增（可能导致回到地面后立刻需要维修才能再次下潜）
- 如果是高层战败，可能获得永久性的「深渊创伤」特质（如：最大HP永久-5%，或背包某个角落格永久封死直到用特殊材料修复）

#### 情感冲击
战败后的工坊画面：
- 魔偶躺在维修台上，战损立绘，面无表情或低声道歉
- 弹出丢失物品列表，物品在玩家面前逐个变为灰烬消散
- 弹出维修账单（通常金额很高）
- 如果好感度较高，她可能会说："对不起主上，我太弱了……那些东西都丢了……"

**设计意图**：失败的惩罚不仅仅是"损失数据"，更要让玩家在**情感上感到心痛**。这种心痛会转化为下一次下潜时"绝对不能再贪"的行为驱动力。

## 六、深渊的层级设计

### 6.1 难度与主题递进

| 层数 | 深度名称 | 环境主题 | 难度特征 | 核心产出 |
|---|---|---|---|---|
| 1~2层 | 浅渊回廊 | 被遗忘的矿道，微弱光线 | 杂兵为主，教学向 | 基础素材、低价值贸易品 |
| 3~4层 | 腐蚀甬道 | 菌丝蔓延的生物区域 | 精英频率增加，开始有对包干涉型怪物 | 中等素材、义体图纸碎片 |
| 5~6层 | 晶化深域 | 大量深渊结晶的异变区域 | 环境侵蚀加重，Boss格子封锁能力强 | 稀有素材、底盘升级核心 |
| 7~8层 | 绝望边境 | 极端扭曲的空间 | 怪物组合极端，多种行为同时出现 | 传说级物品、剧情关键道具 |
| 9~10层 | 深渊核心 | 终极区域 | 真结局相关内容 | 结局触发道具 |

### 6.2 每层的节奏曲线
每层的节点分布遵循一个情绪曲线：

```
起点（安全）→ 几场小战斗（热身）→ 一次事件或宝箱（惊喜/恐惧）
  → 精英战斗（高压高峰）→ 可能的营地（喘息）→ Boss战（决战）
    → 安全区（巨大的释放感 + 撤离/深入的终极抉择）
```

每一层都是一个完整的情绪"小山峰"——从安全到紧张到高潮到释放。玩家选择深入下一层，就是踏入下一座"山峰"的起点。

## 七、深渊的随机性与可控性平衡

### 7.1 地图生成的可控随机
- 地图通过种子随机生成，但**受约束规则限制**：
  - 不会连续出现3个同类型节点
  - 每层至少有1个宝箱节点和1个事件节点
  - Boss节点前必有至少1个营地或宝箱作为缓冲
  - 至少存在2条从入口到Boss的不同路径

### 7.2 战利品掉落的可控随机
- 怪物的掉落池随层级变化而更新（浅层不会掉深层材料）
- 掉落物品的形状是固定的（不会随机变形），玩家可以根据背包空间预判
- 委托物品只在匹配的层级出现

### 7.3 玩家可获取的信息量
| 信息 | 可见性 | 说明 |
|---|---|---|
| 下一个节点的类型 | 可见（图标标识） | 玩家可以做路线选择 |
| 精英怪的具体种类 | 不可见（仅显示⚔️标识） | 保留不确定性 |
| 事件节点的内容 | 不可见（仅显示❓） | 保留冒险感 |
| Boss的种类 | 进入前可见（有前瞻提示） | 允许玩家为Boss做针对性准备 |
| 战利品的具体内容 | 战斗胜利后才可见 | 不会因为预知就做出无风险决策 |

## 八、边界规则

| 情况 | 规则 |
|---|---|
| 玩家在安全区一直不做选择怎么办？ | 时间不消耗。安全区是纯决策节点，不施加时间压力。玩家可以充分权衡 |
| Boss太强了玩家打不过，卡关了怎么办？ | Boss战失败后触发标准的失败惩罚。玩家回到地面后可以通过养成变强后重新挑战。不存在"必须打赢才能回家"的情况——失败也会被强制救回 |
| 玩家裸奔（不带任何装备）进深渊刷碎片 | 允许，但空手无攻击手段，杂兵也打不过。战败的磨损惩罚 + 维修账单远超碎片收益。设计上确保"躺平刷低保"的净收益为负 |
| 安全收纳盒满了但还想保命怎么办？ | 只能在背包中保留绑定物品，其余全凭撤离成功。安全盒容量有限是故意设计的稀缺性 |
| 连续两次下潜同一层，地图一样吗？ | 不一样。每次下潜重新随机生成地图（即使是同一层级）。只有层级主题和敌人池相同 |
