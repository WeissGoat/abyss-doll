# 6. 世界观、剧情与事件反馈体系

## 一、 背景设定蓝图

- **基调与风格：** 融合了微暗黑二次元、蒸汽朋克机械与废土残响的压抑内核。
- **世界地貌：** 
  - **悬丝镇：** 躲避致命“晶化病瘴气”的地表城镇，也是剥削底层的资本堡垒。玩家与魔偶们的日常之地。
  - **深渊：** 瘴气弥漫、藏有古代文明遗迹的巨大地下空间。越往下奖励越高，侵蚀越深劲。
- **角色关系与痛点：** 
  - 主角是一个落魄负债的初出茅庐的“初级魔偶制造师”，深陷资本家和小镇经济周转危机中。
  - 魔偶不仅仅是没有灵魂的战争载具，其核心底层逻辑可能是被注入了曾经生前沾染晶化的女性人类碎片灵魂。当它受到深渊召唤时，便是人性的悲歌触发点。

## 二、 立体四向型剧情触发引擎

为了让世界观和悬疑感并非通过死板的过场演出来传达，采用与机制完全深度挂钩的触发器引擎。

### 1. 强力挤压型（基于系统时间轴）
- **判定核心：** 时间（Time Node / Calendar Event）。
- **设计范式：** 比如“每月倒数第二天催租子”。当这个时间点来临，直接中断一切日常跳出核心主线推进剧情或角色逼债，体现环境冷酷与紧迫感，推进核心全局大主线。

### 2. 重负荷渗透型（基于属性数值 / 隐性养成）
- **判定核心：** 侵蚀度过负荷 (Sanity Overflow) 与 好感度 (Bond Rank)。
- **设计范式：** 当连续三天保持红色的高“侵蚀度”未清理，会触发独有的夜间高压对话（如魔偶神志不清恳求“主上不要再让我下去”等阴暗面发掘）。
- 而好感度达到满级临界点，则会在下一次出动前激活解锁特定好感上限的CG或记忆。

### 3. 深潜里程碑型（基于战斗路线）
- **判定核心：** 玩家当前所在深渊的最大探索层数记录。
- **设计范式：** 游戏世界观最大的谜团位于最底层。所以当玩家第一次通过20层时进入古代安全区内，就自动解锁一段环境叙事的解密信息或录像，把真相散布在节点中。下得越深，知道的越多。

### 4. 战利物品激活型（极具创新的物件叙事逻辑）
- **判定核心：** 物品对象本身挂载。
- **设计范式：** 这是结合背包模式设计的独家彩蛋剧情。
- 深渊有极小概率产出并占位置庞大的“故事残片”（例如一个占 2x2 格子的【陈旧八音盒发条】）。只要玩家能把它忍痛当一块砖头存活带回安全区并在家里点击它，直接解锁关于该女主角的超级隐藏长镜头回忆。这能让玩家为了看剧情情愿少背一点药或武器死扛。

## 三、 深渊最终真结局反馈分支条件

依据系统的最终资源累积倾倒走向：

- **结局 A：资本悬丝（工具线/坏结局）**
  - **达成条件：** 金币充裕能平账、好感度未满，主要升级皆为机体硬件改造。
  - **结局表现：** 在赚取了庞大的财富站稳脚跟后，身边的“女儿”变成了真正无悲无喜的彻底的人形杀人机器，成为悬丝镇权贵的鹰犬。
- **结局 B：逃离暗黑（躺平线/平庸）**
  - **达成条件：** 将底层资源兑换完毕以行贿的形式跑路，没打穿到底层探寻真相。
  - **结局表现：** 带着魔偶开了普通的隐居小卖部苟活，虽然有温馨日常但体内深渊侵蚀暗门永远无法根除。
- **结局 C：破晓凝结（真结局/完美线）**
  - **达成条件：** 直通深渊最底层节点，解决大型心智BOSS，并使所有属性达到同步/好感度全解锁状态。
  - **结局表现：** 利用羁绊护盾引爆奇迹抵挡瘴气，帮助魔偶彻底融合核心脱去机械形态，血肉与记忆均得到重组新生，挣脱诅咒。
