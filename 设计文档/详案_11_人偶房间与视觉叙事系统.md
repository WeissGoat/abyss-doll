# 详案_11：人偶房间与视觉叙事系统 (GDD_11)

> **定位说明：** 人偶房间是整个游戏中唯一一个"没有任何压力"的空间。深渊有死亡压力，开店有经济压力，日程有时间压力——而房间的存在，是为了让玩家在所有压力之间获得一个情感缓冲区。它同时也是整局游戏的"视觉日记"，忠实记录着玩家与魔偶共同经历的一切。
>
> **开发优先级：** 低（后期拓展模块）。本模块不影响核心循环运转，但对游戏的情感深度和二次传播价值有重大贡献。建议在核心循环（战斗/探索/经营）稳定后再着手实现。

---

## 一、房间的定位与设计哲学

### 1.1 为什么需要一个房间？

在搜打撤+模拟经营的高压循环中，玩家持续面对"下不下潜""卖不卖""修不修"的抉择焦虑。如果整个游戏全程都在做决策、都在承受后果，玩家会疲劳、会麻木。

房间的作用是**让玩家记起"她不只是一个战斗工具"**。

- 深渊中的她是一台背负背包的战斗机器
- 商店中的她是一个招揽客人的看板娘
- 但在房间里，她只是一个坐在窗边发呆的女孩

### 1.2 设计三原则

| 原则 | 说明 |
|---|---|
| **零压力** | 房间里不存在任何资源消耗、倒计时、效率计算。进入房间不消耗AP |
| **纯表达** | 房间的布置只影响表现层（立绘、氛围、微量被动效果），不影响核心战斗数值 |
| **缓慢积累** | 房间不是一次装修完的，每一件家具都来自游戏中的不同阶段，是"一路走来"的物证 |

## 二、房间的空间结构

### 2.1 房间布局
房间是一个固定视角的横版2D场景，背景为工坊楼上的一间阁楼。

**空间分区**（从左到右）：

| 区域 | 占比 | 核心内容 |
|---|---|---|
| 窗边区 | 约25% | 窗户 + 窗台。天气/时间变化在此呈现 |
| 生活区（中央） | 约40% | 床铺 + 地毯 + 主要家具摆放位置 |
| 陈列区 | 约25% | 架子/展柜，用于摆放纪念品和礼物 |
| 角落工位 | 约10% | 小桌子 + 台灯，魔偶独处时的默认位置 |

### 2.2 魔偶在房间中的表现
- 进入房间时，魔偶会有一个自然的待机动作（坐在窗边看外面、趴在桌上打瞌睡、站在展柜前端详某件礼物）
- 待机动作不是随机的——**由当前状态决定**：
  - SAN值充足 + 好感度高 → 看窗外、轻轻哼歌
  - SAN值偏低 → 蜷缩在床角、抱着枕头
  - 刚从深渊回来（高磨损） → 坐在床上擦拭自己的关节
  - 好感度极高 → 看到玩家进来时抬头微笑
  - 好感度极低 → 背对门坐着，不回头

## 三、家具系统

### 3.1 家具的本质
家具不是"装备"。它不提供ATK+10这种数值。家具是**氛围载体**——影响的是魔偶在房间中的行为表现、休息效率，以及玩家的视觉体验。

### 3.2 家具分类

#### A. 功能型家具 — 影响休息效率

这类家具影响日程中「休息」行动的回复量，属于微量被动加成。

| 家具 | 效果 | 获取方式 | 价格区间 |
|---|---|---|---|
| **破旧行军床**（初始） | 休息行动恢复15点SAN | 初始自带 | 免费 |
| **棉被铁架床** | 休息行动恢复20点SAN | 小镇家具店购买 | 500G |
| **天鹅绒软床** | 休息行动恢复30点SAN | 小镇家具店购买 | 2000G |
| **深渊蚕丝大床** | 休息行动恢复40点SAN + 每日自动恢复5点侵蚀度 | 需要深渊素材「深渊蚕丝」+ 3000G | 特殊制造 |

| 家具 | 效果 | 获取方式 |
|---|---|---|
| **简易书架**（初始） | 无效果，纯装饰 | 初始自带 |
| **炼金文献架** | 工坊特训时属性获取效率+10% | 法师塔声望奖励 |
| **深渊解析仪** | 查看仓库中物品的隐藏属性（如未鉴定遗物的真实价值预估） | 机械工坊声望奖励 + 2000G |

#### B. 氛围型家具 — 改变房间视觉与魔偶行为

这类家具不影响任何数值，只改变房间的美感和魔偶在房间中的互动内容。

| 家具 | 视觉效果 | 魔偶行为变化 |
|---|---|---|
| **暖光壁灯** | 房间从冷色调变为暖橘色 | 增加"看书"待机动作 |
| **留声机** | 背景加入轻柔的音乐盒BGM | 增加"闭眼听曲"待机动作 |
| **阁楼望远镜** | 窗户视野拉远，能看到小镇全貌 | 增加"用望远镜看远方"待机动作 |
| **盆栽/花瓶** | 窗台出现植物，随时间缓慢生长 | 增加"浇花"待机动作 |
| **地毯** | 地板从木纹改为柔软地毯 | 增加"坐在地上翻看图鉴"待机动作 |

#### C. 纪念型家具 — 不可购买，只由经历产生

这类家具无法在商店购买，它们在**特定剧情/里程碑发生后自动出现在房间中**。玩家不能选择放或不放——它们是"这段经历真实发生过"的烙印。

| 纪念物 | 触发条件 | 房间表现 |
|---|---|---|
| **墙上的裂痕** | 首次在深渊中战斗失败 | 阁楼墙角出现一道裂缝，永远修不好 |
| **挂墙的旧武器** | 替换掉使用最久的第一把武器后 | 那把旧武器被挂在墙上作为纪念 |
| **床头的旧照片** | 好感度首次达到Lv.5 | 床头出现一张模糊的合影（剪影式，不画脸） |
| **窗台的干花** | 第一次送花类礼物 | 窗台出现一束永远不会枯萎的干花 |
| **角落的积灰木箱** | 触发经济破产事件后又恢复 | 角落出现一个破旧木箱，提醒那段最困难的日子 |
| **书桌上的日记本** | 累计游玩时间超过5小时 | 书桌上出现日记本。点击可回顾"到目前为止的关键事件时间线" |
| **天花板的星星贴纸** | 好感度达到Lv.8以上 | 天花板角落出现荧光星星贴纸——世界观包装为"她偷偷贴上去的" |

## 四、陈列系统——房间即视觉日记

### 4.1 陈列区的设计思路
房间右侧的陈列区有一组展柜/架子，上面可以摆放**玩家主动选择展示的物品**。

这不是仓库，不是货架——这是一个"纪念博物馆"。玩家把某件物品放在展柜上时，不是在"存放"它，而是在说：**"这个东西对我来说有意义。"**

### 4.2 陈列规则
- 展柜有 **6~8个展位**
- 可以放置任何在仓库中的物品实例（不会被消耗掉）
- 放上展柜的物品**依然在仓库中**，不影响其使用。展示的是"复制品/剪影"
- 每件展示品下方自动标注获取时间和来源（"第3月·深渊第5层·弑龙者之牙"）

### 4.3 魔偶与陈列品的互动
魔偶在待机时偶尔走到展柜前端详某件展品。不同展品会触发不同的魔偶微反馈：

| 展品类型 | 魔偶反馈 |
|---|---|
| 她使用过的旧武器 | 拿起来比划一下，然后放回去 |
| 打败Boss掉落的战利品 | 盯着出神，想起那场战斗 |
| 玩家送过的情感礼物 | 轻轻摸一下，露出微笑 |
| 诅咒物品（如果玩家故意展示） | 皱眉退后一步，但不说什么 |
| 值钱的贸易品（玩家舍不得卖） | 歪头表示困惑——"为什么不卖掉？" |

### 4.4 陈列的隐藏机制
当展柜中同时摆放了**满足特定组合条件**的物品时，会触发隐藏对话事件：

| 组合条件 | 触发对话 |
|---|---|
| 第一把武器 + 当前最强武器 | "从这把生锈的小刀到现在……我们走了好远呢。" |
| 3件以上深渊不同层的战利品 | "每一件都是从那个黑暗中带回来的证据。我们真的到过那么深的地方。" |
| 情感类礼物 ≥ 3件 | 根据好感度等级触发不同的感谢对话 |
| 只有贸易品（全是值钱东西） | "主上……你是把这里当金库了吗？"（轻微掉好感+增加「财迷」特质概率） |

## 五、窗户——时间与世界的呼吸口

### 5.1 窗外的场景
房间左侧的窗户是连接"房间"和"外部世界"的唯一通道。窗外的景色会根据**当前时间和游戏进度**实时变化。

#### 时间变化
| 时段 | 窗外表现 |
|---|---|
| 上午（AP①） | 晨光穿入房间，柔和的金色光线 |
| 下午（AP②） | 明亮的日光，偶尔可见小镇的烟囱冒烟 |
| 夜晚（AP③） | 路灯和星空，房间只靠壁灯/台灯照明 |

#### 天气变化（纯视觉，不影响系统）
| 天气 | 触发条件 | 表现 |
|---|---|---|
| 晴天 | 默认 | 蓝天白云 |
| 雨天 | 深渊返回后的第一个夜晚（如果战损严重） | 窗上雨滴，雨声环境音 |
| 雪天 | 游戏后期月份（第10~12月） | 窗外飘雪，房间色调变冷 |
| 红月 | 即将触发深渊异动事件前夜 | 窗外月亮变为暗红色，预兆不祥 |

#### 剧情变化
| 事件 | 窗外变化 |
|---|---|
| 小镇传闻:"炼金工坊爆炸" | 窗外远处可见烟柱 |
| 玩家经济破产 | 窗外小镇灯火更暗淡 |
| 完成真结局前置条件 | 窗外地平线出现微弱的深渊光柱 |

### 5.2 窗户的叙事意义
窗户不只是美术装饰。它让玩家即使在房间里也能感受到"世界在继续运转"。当下雨的夜晚，魔偶坐在窗边静静看着雨滴从玻璃上滑落——这一幕不需要任何文字就能传达情感。

## 六、房间中的交互

### 6.1 可交互的行为

房间中玩家可以做的事情极少——这是故意的：

| 行为 | 效果 | 说明 |
|---|---|---|
| **点击魔偶** | 触发简短反馈对话（1~2句） | 对话内容由当前状态池随机抽取 |
| **点击展柜** | 打开陈列管理界面 | 添加/移除展示品 |
| **点击窗户** | 切换为窗外全景视角（纯观赏） | 看看小镇的全貌，几秒后自动切回 |
| **点击床铺** | 如果当前为夜晚且魔偶SAN低 → 触发就寝对话 | 可能解锁特殊晚安对话（好感度足够高时） |
| **什么都不做，停留30秒以上** | 魔偶开始执行长待机动画 | 可能出现稀有的隐藏小动作（伸懒腰、对着镜子比划表情、偷偷看玩家） |

### 6.2 房间不消耗AP
进入房间、在房间里待多久，都不消耗行动点。玩家可以在分配完日程后、在任何行动的间隙进入房间。

> **设计意图**：如果进入房间也消耗AP，玩家会把它当作"不划算的选项"来优化掉。让它免费，才能确保它被视为"休憩"而非"任务"。

## 七、房间的情绪叙事时间线

### 7.1 房间随游戏阶段的变化
房间不是静态的。随着游戏推进，它会一点一点地从空荡变得丰满，最终成为"你和她共同生活的痕迹"。

| 游戏阶段 | 房间状态 |
|---|---|
| **开局（第1月）** | 空荡荡的阁楼，只有破旧行军床、简易书架和一盏快坏的灯。窗户上有灰。魔偶坐在角落 |
| **前期（第2~3月）** | 玩家买了第一件家具。魔偶开始有新的待机动作。窗户被擦干净了 |
| **中期（第4~7月）** | 展柜上开始有东西了。墙上可能已经有了裂痕（首次战败的记忆）。也许有了干花和暖光灯 |
| **后期（第8~11月）** | 房间开始有了"家"的感觉。魔偶在房间里的行为更自然放松（前提是好感度正常发展） |
| **终局前夜（第12月最后一周）** | 根据结局走向，房间呈现不同的氛围——温馨的满足感 / 空洞的虚无感 / 病态的占有排列 |

### 7.2 结局前的房间特殊表现
在最终月的最后几天，房间会根据即将触发的结局方向产生细微变化：

| 结局方向 | 房间最终状态 |
|---|---|
| **真结局（伴侣线）** | 展柜上多了一朵新鲜的花（不是干花），窗外阳光特别明亮 |
| **无情兵器线** | 展柜上只有武器和战利品，没有任何礼物。房间极其整洁但冰冷 |
| **经济破产线** | 房间的家具被债主搬走了一部分，空荡荡的痕迹 |
| **魔偶崩溃线** | 房间一片狼藉，物品散落一地，窗户碎裂 |
| **病娇隐藏线** | 展柜上所有物品被重新排列成某种祭坛般的图案。窗户被封死 |

## 八、房间与其他系统的微弱关联

### 8.1 关联原则
房间对其他系统的影响必须保持**微弱且正向**——绝不能让"不装修房间"的玩家感到自己在数值上严重落后。

| 关联系统 | 影响方式 | 影响强度 |
|---|---|---|
| **日程·休息行动 (GDD_07)** | 床铺品质影响SAN恢复量 | 微弱（差距在15~40点） |
| **日程·特训行动 (GDD_08)** | 书架品质影响特训效率 | 微弱（+10%） |
| **好感度 (GDD_03)** | 房间舒适度影响"陪伴"行动的好感获取效率 | 微弱（好家具+5%效率） |
| **特质 (GDD_09)** | 展柜内容组合可能触发隐藏对话，偶尔影响特质获取/消除 | 极微弱 |

### 8.2 为什么不做更多关联？
如果房间装修对战斗力有显著影响（比如"高级床 → 全武器攻击力+15%"），那装修就变成了一个强制投资项目。玩家会用Excel算最优装修路线，而不是凭感情选家具。

**房间的一切决策都应该出自情感，而非理性计算。**

## 九、边界规则

| 情况 | 规则 |
|---|---|
| 纪念型家具能不能移除/隐藏？ | 不能。"墙上的裂痕"就在那里，永远提醒你那次失败。这是叙事设计的一部分 |
| 展柜中的物品被卖掉了怎么办？ | 展示品从展柜中消失，展位空出。但物品曾经被展示过的记录保留在日记本中 |
| 房间家具有上限吗？ | 每个区域的家具位是固定的（如床铺位只有1个、架子位只有1个）。买新家具替换旧的，旧的自动回收（不退款）|
| 多周目时房间继承吗？ | 不继承。新周目是新的阁楼、新的魔偶。前一局的所有痕迹都留在了前一局的记忆里——但日记本系统会保留一份"往期回顾"供玩家在主菜单查看 |
| 玩家从来不进房间会怎样？ | 不影响核心游戏进程。但会错过隐藏对话、展柜组合触发事件。魔偶的部分特质（如「独立意识」）的获取可能与房间互动频率有关 |
