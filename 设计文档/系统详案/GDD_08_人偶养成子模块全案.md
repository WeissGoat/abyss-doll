# 详案_08：人偶养成子模块全案 (GDD_08)

> **定位说明：** GDD_03 定义了魔偶的"灵魂"（好感双轨 + 特质引擎）。本文档定义魔偶的"肉身"——玩家花钱、花材料、花时间去改造和维护她的全部养成子模块。它们共同构成玩家在局外的核心资源消耗出口。
>
> **与其他文档的关系：**
> - 底盘重构 → 直接改变 GDD_01（背包网格）的形态画布
> - 义体插件 → 为 GDD_06（物品系统）中的装备标签提供局外加成
> - 机体维护 → 是 GDD_07（日程系统）中的核心AP消耗项
> - 心智天赋树 → 为 GDD_02（搜打撤）的失败提供低保补偿
> - 换装涂装 → 是 GDD_04（经济系统）的纯金币回收池

---

## 一、总体架构：五大养成子模块

人偶的养成被拆分为五个独立模块，各自消耗不同的资源，产出不同方向的成长。玩家不可能同时推进所有模块——资源的稀缺性逼迫玩家在养成方向上做选择。

| 模块 | 一句话定义 | 消耗资源 | 产出方向 |
|---|---|---|---|
| ① 底盘重构 | 换一副新骨架 | 巨额金币 + Boss核心 | 改变背包形状与构筑底层规则 |
| ② 义体插件 | 穿戴固定配件 | 深渊素材 + 少量金币 | 局外固定属性加成 |
| ③ 心智天赋树 | 点亮灵魂印记 | 记忆碎片（失败低保） | 被动能力与容错率 |
| ④ 机体维护 | 修复与保养 | 金币 + 净化结晶 | 消除负面状态，维持可用性 |
| ⑤ 换装涂装 | 换衣服 | 纯金币 | 微量情感加成 + 视觉变化 |

**设计黄金律**：前三个模块（底盘/义体/天赋树）是"做加法"的正向成长，第四个模块（机体维护）是"抗减法"的损耗对冲，第五个模块（换装）是纯粹的情感投资和金币回收口。

## 二、模块①：底盘重构（Chassis Reconstruction）

### 2.1 定位
底盘是魔偶最核心、最昂贵、更换频率最低的"主体骨架"。它直接决定了局内背包的**网格形状、大小和底层被动规则**——可以理解为"换了一个人"。

### 2.2 初始选择
游戏开局时，玩家选择的魔偶自带一副初始底盘。不同魔偶的初始底盘差异很大，引导出完全不同的首局体验（详见下一节的底盘图鉴）。

### 2.3 更换方式
底盘不叠加升级——是整体更换。玩家在工坊中将当前底盘替换为新底盘，旧底盘回到仓库（可在下一周目继承为解锁选项）。

**更换限制**：
- 每次更换底盘后，魔偶需要 **3天适应期**（这3天内下深渊时，背包中随机1~2个边缘格子处于"不稳定"状态，可能在战斗中临时抖动偏移——包装为"新骨架还没磨合"）
- 适应期间可以通过安排「机体维护」来缩短1天

### 2.4 底盘图鉴

#### 【标准骑士底盘】— 新手通用
- **初始网格**：5×5 正方形
- **被动规则**：无特殊规则，充能速度均衡
- **解锁条件**：初始可选
- **设计意图**：最传统的拼图体验，容错率高，适合学习期

#### 【超载供能底盘】— 重火力流
- **初始网格**：6×6 大网格，但中心有2×2区域为「高温核心」（无法放置普通物品）
- **被动规则**：放在高温核心**周围8格**的武器攻击力翻倍，但每次攻击消耗魔偶少量理智值
- **解锁条件**：从「机械工坊」势力购买图纸（需声望3级 + 3000G） + 击杀深渊4层Boss并带回「深渊熔炉核心」
- **设计意图**：围绕中心的"禁区"做布局，用冷却管道和隔热板衍生出独特的空间规划

#### 【影袭底盘】— 敏捷刺客流
- **初始网格**：3×6 狭长矩形
- **被动规则**：背包中每有1个空格，所有武器冷却时间-3%（鼓励轻装出行）
- **解锁条件**：完成「冒险者公会」连锁委托 + 3根「深渊蜘蛛腿」
- **设计意图**：极小的宽度使得横向摆放大型武器几乎不可能，逼迫玩家走小型快攻武器路线；空格加速的规则与"搜打撤"矛盾极强——捡东西越多，战斗力越低

#### 【共鸣回路底盘】— 标签羁绊流
- **初始网格**：十字形（中心3×3 + 四个方向各延伸2格）
- **被动规则**：背包中每凑齐3件同标签物品，触发一次全局共鸣效果（该标签的专属Buff）
- **解锁条件**：从「法师塔」学习蓝图（需声望4级 + 2000G） + 2块「共鸣晶簇」
- **设计意图**：异形网格造成大量"边角空间"，但共鸣奖励鼓励玩家堆叠同类物品，用标签数量弥补形状劣势

#### 【看板娘底盘】— 经营特化
- **初始网格**：4×4 超小方格
- **被动规则**：战斗能力极差；但在商店经营时，自带「空间折叠」能力——商店货架的展示面积+50%
- **解锁条件**：累计经营收入达到某个里程碑
- **设计意图**：把"下深渊"的难度推到极致（你只能带一把匕首和两瓶药），迫使玩家走"靠经营暴富 → 用钱砸出义体补战力"的纯经营路线

### 2.5 底盘与结局的关联
月末的阶位进化判定中，魔偶当前装备的底盘类型是结局分支的参考变量之一。例如：
- 全程使用「超载供能底盘」 + 机能属性最高 → 倾向「重装机甲/无情兵器」结局分支
- 全程使用「看板娘底盘」 + 魅力属性最高 → 倾向「完美看板娘/金牌掌柜」结局分支

## 三、模块②：义体插件（Prosthetic Plugins）

### 3.1 定位
义体插件是**不占用背包格子的局外固定装备**。它们提供稳定的被动数值加成或功能拓展。类似传统RPG的"装备栏"，但效果全部倾向于辅助背包战斗或经营，而非直接提供暴力攻击数值。

### 3.2 插件槽位
魔偶身上有 **5个固定的义体槽位**，每个槽位只能装一件对应类型的插件：

| 槽位 | 部位 | 主要加成方向 | 示例效果 |
|---|---|---|---|
| 头部传感器 | 感知/侦察 | 深渊地图信息 | "提前显示下一个未知节点是陷阱还是宝箱" |
| 右臂模组 | 攻击/操作 | 武器加成 | "所有`[枪械]`标签武器冷却时间-5%" |
| 左臂模组 | 防御/功能 | 生存加成 | "每次撤离时自动绑定背包中价值最高的1件物品" |
| 动力腿 | 移动/负重 | 背包辅助 | "背包最外圈格子的物品耐久消耗减半" |
| 能量泵 | 核心/全局 | 三维属性 | "机能+10 / 算力+10 / 同步率+10（选其一）" |

### 3.3 插件获取方式
- **制造**：在工坊中消耗深渊素材 + 金币制造。需要先拥有对应的「图纸」。
- **图纸来源**：
  - 势力声望奖励（机械工坊声望每级解锁新图纸）
  - 深渊遗物研究转化（详见GDD_06遗物章节）
  - 部分由小镇商店在特殊事件日出售

### 3.4 插件品级
插件分为三个品级，差异在于效果强度和素材消耗：

| 品级 | 制造消耗 | 效果强度 | 来源 |
|---|---|---|---|
| 基础（白） | 普通素材（机械残骸等） + 少量金币 | 低 | 初始即有图纸 |
| 强化（蓝） | 稀有素材（晶化心脏、蜘蛛腿等） + 中等金币 | 中 | 势力声望奖励 |
| 极致（紫） | Boss掉落核心 + 大量金币 | 高 | 遗物研究 / 高级势力声望 |

### 3.5 设计原则
- 义体**绝不直接加攻击力/血量的裸数值**（那是背包里的武器该做的事）
- 义体的效果必须是"改变规则"或"拓展可能性"，而非"数值暴涨"
- 目的是让玩家通过义体的选择来**延伸和补强底盘的构筑思路**

## 四、模块③：心智天赋树（Memory / Mental Matrix）

### 4.1 定位
心智天赋树是深渊探索**失败后的补偿机制（低保）**，同时也是魔偶"灵魂觉醒"的可视化表征。它确保玩家即使在深渊中全军覆没，也能带回微弱但永久的成长，避免"白给了一局什么都没得到"的极端挫败感。

### 4.2 核心资源：记忆碎片
- 记忆碎片是心智天赋树的**唯一消耗资源**
- 获取方式：
  - **深渊探索**：无论成败，每进入一个节点都会自动获取少量记忆碎片（非实体物品，不占背包格子，不可丢弃）
  - **成功撤离**：额外获得一批数量可观的碎片（按到达深度计算）
  - **战斗失败**：保留已累积的碎片（这是失败后唯一不被清零的资源）
- **设计意图**：记忆碎片是"经验的沉淀"，不受背包空间约束。即使玩家裸奔进深渊送死，也会因为"走过的路"而积累少量碎片。但成功撤离获得的碎片远多于失败，依然鼓励认真游玩。

### 4.3 天赋树结构
天赋树分为**三条分支**，对应游戏的三大核心循环：

#### 分支A：战斗直觉
面向深渊战斗的容错与增益。

| 节点层级 | 节点名 | 效果 |
|---|---|---|
| 第1层 | 本能闪避 | 每层深渊的第一次受击伤害减半 |
| 第2层 | 背包直觉 | 战斗中首次打开子弹时间时额外延长1秒 |
| 第3层 | 深渊感应 | 进入安全区时，自动显示下一层的敌人构成概况 |
| 第4层 | 战损经验 | 战斗失败后背包中物品丢失比例从100%降至80%（保留20%最便宜的物品） |
| 终极节点 | 绝地求生 | 每次下潜获得1次额外的"保底撤离机会"——在任意战斗失败时可选择强制撤退到上一个安全区（代价：丢失本层所有新获取物品） |

#### 分支B：商业头脑
面向模拟经营的效率与视野。

| 节点层级 | 节点名 | 效果 |
|---|---|---|
| 第1层 | 敏锐估价 | 仓库中的物品自动显示当前市场售价（无需去店铺查看） |
| 第2层 | 甜嘴折扣 | 在小镇商店购买物品时享受5%折扣 |
| 第3层 | 看板气质 | 开店经营时客流量基础值+20% |
| 第4层 | 情报耳朵 | 传闻物价变动提前1天在情报板上显示（比正常周结算早1天） |
| 终极节点 | 天才掌柜 | 开店时偶尔触发"捡漏事件"——自动识别出某件深渊垃圾实际是伪装的古董，卖价暴增10倍 |

#### 分支C：深渊抗性
面向SAN值/侵蚀度的抗压能力。

| 节点层级 | 节点名 | 效果 |
|---|---|---|
| 第1层 | 冥想呼吸 | 每次进入安全区时自动恢复5点SAN值 |
| 第2层 | 精神屏障 | 侵蚀度每层增长量-10% |
| 第3层 | 瘴气适应 | 背包中被`[诅咒]`物品辐射的格子数量上限减少1格 |
| 第4层 | 理智锚定 | SAN值不会降到10以下（保留最低10点底线，避免0SAN的极端幻觉惩罚） |
| 终极节点 | 深渊之女 | 侵蚀度不再产生背包格子封锁效果（但高侵蚀度的其他惩罚如幻觉立绘变化仍存在） |

### 4.4 点亮规则
- 每个节点需要消耗一定数量的记忆碎片（越深层消耗越多）
- 必须从第1层开始依次解锁，不可跳级
- 三条分支独立，记忆碎片在三条分支中自由分配
- 天赋树进度**在局间不重置**——在多周目中保留，作为Meta-Progression（元进度）的一部分

### 4.5 天赋树与结局的关联
最终结局判定时，三条分支中哪条的深度最高，会影响结局的"底色"：
- 战斗直觉最深 → 结局中魔偶偏向"战士/守护者"气质
- 商业头脑最深 → 结局中魔偶偏向"商人/独立个体"气质
- 深渊抗性最深 → 结局中魔偶偏向"觉醒者/超越人偶"气质

## 五、模块④：机体维护与保养（Maintenance）

### 5.1 定位
维护不是"让魔偶变强"，而是"阻止魔偶变弱"。它是游戏中**最核心的资源消耗口**——玩家赚来的每一分钱，首先要拿出一部分来"交维护税"，剩下的才能投资其他模块。

如果说底盘和义体是"做加法"的快感，维护就是"做减法"的紧迫感。没有这个模块，金币会无限膨胀，游戏经济循环会在中期彻底崩溃。

### 5.2 维护的两个层面

#### 层面A：物理修复（磨损度）
- **损耗来源**：深渊中每次受到攻击、战斗失败、或携带了某些"腐蚀类"物品
- **不修复的惩罚**：
  - 轻度磨损（磨损度30%~60%）：背包边缘1~2个格子出现"裂纹"状态——放在裂纹格上的易碎消耗品（药水等）在战斗中有概率碎裂流失
  - 重度磨损（磨损度60%以上）：背包边缘格子直接封锁，无法放置任何物品
  - 极端磨损（磨损度90%以上）：日程中「下深渊」选项被强制锁定（详见GDD_07）

- **修复消耗**：
  - 基础修复：少量金币 + 1AP → 降低30%磨损度
  - 精密修复：中等金币 + 「精密零件」 + 1AP → 降低60%磨损度

#### 层面B：瘴气净化（侵蚀度）
- **损耗来源**：深渊中每推进一个节点、每面对恐怖类敌人、每携带`[高侵蚀]`标签物品
- **不净化的惩罚**：
  - 中度侵蚀（50%~70%）：深渊中随机出现幻觉色块占据空格子
  - 高度侵蚀（70%以上）：下深渊时会出现随机走错方向的情况（详见GDD_07）
  - 极端侵蚀（接近100%）：触发「崩溃事件」，强制占用全部AP

- **净化消耗**：
  - 基础净化：金币 + 「净化结晶」(深渊浅层素材) + 1AP → 降低40%侵蚀度
  - 深度净化：大量金币 + 「高纯净化结晶」(深渊深层素材) + 1AP → 侵蚀度归零

### 5.3 维护的交互包装
维护不只是一个"点击按钮扣钱"的操作。它是连接玩家情感和数值的桥梁：

- **物理修复的包装**：玩家用焊枪修补魔偶的战损部位，伴有火花特效和魔偶的低声忍痛语音
- **瘴气净化的包装**：玩家用刷子在魔偶身上擦洗黑色瘴气粘液，擦干净的瞬间立绘焕然一新
- **高级保养品的差异**：
  - 用廉价工业机油修复（回复相同量的磨损，但**降低少量好感度**——"冰凉的机油让她觉得自己只是个机器"）
  - 用昂贵的带花香护理液修复（回复相同量的磨损，且**增加好感度**——"温热的液体伴有淡淡花香，她露出了难得的放松表情"）

> **设计意图**：即使是"修机器"这种功能性操作，也通过保养品的高低档次，把"花钱 ↔ 感情"的博弈嵌进去。玩家不缺资源时，自然会选更贵的；但月底租金来了、兜里只剩最后几百块时，那瓶廉价机油就变成了一次深深的内疚。

### 5.4 维护作为经济回收池
维护费用随深渊层数推进而自然递增——越深层的怪造成的磨损/侵蚀越严重，修复所需的材料越高级越贵。这保证了：

- 前期（浅层探索）：维护费微乎其微，玩家能积累初始资金
- 中期（中层探索）：维护费成为固定支出的重要一环
- 后期（深层探索）：维护费可能占据总收入的30%~50%，与租金共同构成双重经济压力

## 六、模块⑤：换装与涂装（Wardrobe & Customization）

### 6.1 定位
换装系统是**纯粹的金币回收池 + 情感服务模块**。它不提供任何有意义的战斗数值加成，存在的唯一目的是：
1. 让深渊里赚来的血汗钱有一个"心甘情愿烧掉"的出口
2. 满足二次元玩家对角色定制的情感需求
3. 用视觉变化来标记养成历程（"她身上的每一件衣服，都是我从深渊里拼命赚来的"）

### 6.2 换装分类

| 类别 | 穿着场景 | 微量效果 | 获取方式 |
|---|---|---|---|
| 战斗装甲 | 深渊探索时的立绘 | 无数值加成（纯视觉） | 解锁底盘时自动附带对应款式 |
| 日常便服 | 小镇/工坊/开店时的立绘 | 穿着便服开店时，客流魅力值+5% | 小镇成衣店购买（纯金币） |
| 特殊服装 | 取决于服装类型 | 穿着特定服装时，保养恢复效率+10% | 特殊事件奖励 / 好感度里程碑解锁 |
| 配饰 | 全场景 | 无 | 好感度升级自动解锁（免费） |

### 6.3 设计原则
- 服装**永远不影响战斗力**——如果穿好看的衣服能加攻击，玩家就会把换装当装备栏来算，失去情感价值
- 便服的微量经营加成足够制造"买一件总比没有好"的心理，但不足以成为"不买就亏"的强制需求
- 好感度里程碑自动解锁的配饰，是对玩家"养成历程"的纪念品——不花钱，纯感情

### 6.4 价格定位
便服的价格应该处于**"有点贵，但不是买不起"的微妙位置**——大约等于一次中等难度深渊探索的净收入。

玩家在决定是否给她买裙子时的内心计算公式应该是：
> "这条裙子1500G……相当于我冒着半死的风险下一趟深渊3层能赚到的钱……但看到她穿上之后笑了一下，好像又值了。"

## 七、五大模块的资源流转总览

```
┌─────────────────────────────────────────────────────┐
│                 深渊探索 (资源产出端)                    │
│   产出→ 金币(卖战利品) / 素材 / 碎片(自动) / Boss核心      │
└───────────┬────────────┬───────────┬────────────┬────┘
            │            │           │            │
            ▼            ▼           ▼            ▼
   ┌────────────┐ ┌──────────┐ ┌──────────┐ ┌────────┐
   │ ④机体维护    │ │①底盘重构  │ │②义体插件   │ │⑤换装   │
   │ (金币+素材)  │ │(金币+核心)│ │(素材+金币) │ │(纯金币) │
   │ 最优先消耗   │ │ 最昂贵    │ │ 日常消耗   │ │情感消耗 │
   └────────────┘ └──────────┘ └──────────┘ └────────┘
            ▲
            │ (失败也能获得)
   ┌────────────┐
   │ ③心智天赋树   │
   │ (记忆碎片)    │
   │ 独立补偿闭环  │
   └────────────┘
```

**资源消耗优先级**（玩家的自然行为）：
1. **机体维护** → 因为不修就无法下深渊，是强制消耗
2. **租金** → 因为不交就破产Game Over（详见GDD_07）
3. **义体插件** → 性价比最高的常规投资
4. **底盘重构** → 里程碑式的巨额消耗，需要长期积攒
5. **换装** → 有余钱才会考虑的"奢侈品"

## 八、养成节奏与阶段体验

### 前期（1~3月）：生存导向
- 玩家刚接手一个破旧工坊，魔偶只有初始底盘和白色义体
- 金币主要用于交租和基础维护
- 心智天赋树只点了1~2层浅节点
- 体验关键词：**挣扎、学习、基础循环确立**

### 中期（4~8月）：投资分化
- 玩家开始有余钱考虑更换底盘或强化义体
- 势力声望陆续解锁新图纸，玩家面临"先升底盘还是先做义体"的分支决策
- 天赋树开始成形，三条分支中某一条到达第3~4层
- 给她买了第一件衣服
- 体验关键词：**选择、分化、个人风格成型**

### 后期（9~12月）：极限压榨与告别
- 租金和维护费双重挤压，玩家每一分钱都在精打细算
- 底盘已经定型，义体基本成形，天赋树某一分支接近终极节点
- 结局条件开始收束，玩家的养成偏向决定了魔偶的最终形态
- 体验关键词：**极限博弈、情感高潮、尘埃落定**

## 九、边界规则与特殊交互

| 情况 | 规则 |
|---|---|
| 更换底盘后，原来背包里不合形状的物品怎么处理？ | 更换底盘在工坊中进行，此时背包为空（所有物品都在仓库里）。新底盘的适配过程不涉及物品迁移 |
| 义体插件能不能拆下来换给另一个魔偶（多周目）？ | 义体与当前魔偶绑定，本局无法转让。新周目的新魔偶需要重新制造（但图纸保留） |
| 天赋树三条分支都点满了会怎样？ | 不可能同时点满——总碎片量经过计算，一局的上限约够点满1.5条分支。这迫使玩家做取舍 |
| 完全不做维护，磨损度和侵蚀度叠满了会怎样？ | 磨损度100%：背包边缘3~4格永久封锁直到修复。侵蚀度100%：触发魔偶崩溃事件（详见GDD_07），强制停摆 |
| 买了一件衣服后能退吗？ | 不能退货。这是故意设计——让"买衣服"成为一个需要权衡的消费行为 |
| 保养时选了廉价机油，但好感度已经快冰点了怎么办？ | 廉价保养仍然会掉好感（哪怕只掉一点），可能导致好感度跌入冰点区。此时能修复好感的唯一途径是：花更多的时间和金币去做陪伴行动（详见GDD_03/GDD_07） |
