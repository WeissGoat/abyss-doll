# 详案_04：小镇循环与经济物价波浪模型 (GDD_04)

> **定位说明：** 游戏的外围压迫器与驱动引擎。详细设计金币如何产生、如何消耗，以及资本是如何逼迫手无寸铁的魔偶再次跳下深渊的。
> 
> **重要变更记录：** 本文档已废弃原有的"手动开店与逐个顾客议价"设计，重构为基于心流的高效"三步阶段结算制"（倾倒箱 + VIP橱窗），将繁琐的陈列出售过程通过自动化动画展现，玩家保留高价值的拼图策略决策。

---

## 第一部分：营业结算环节（核心三步走）

营业环节不再是消耗行动时间的独立内容，而是每日夜晚必须经历的**“阶段结算制”流程**，这是玩家从高压的深渊（或一天的休整）后释放压力、获得正向反馈的多巴胺时刻。整个营业结算流程干脆利落地分为三步：

### 1. 战利品入库与一键倾倒（体验"大丰收"的解压感）
- **UI布局：** 屏幕中央是刚刚从深渊带回的塞得乱七八糟的“战斗背包”，右侧是“店铺倾倒箱”（超大网格/堆叠栏区）与“永久仓库”。
- **操作逻辑：** 玩家使用**“一键倾倒”**按钮，把所有非锁定/非装备的杂物，像倒垃圾一样全部扫进店铺倾倒箱。这些普通资源不再需要玩家单独议价，将在下一步以极低的操作成本转化为基础金币收益。

### 2. 镇店橱窗与网格拼图（体验"暴利变现"的策略感）
这是此阶段唯一需要动脑的地方，也是将游戏核心的"网格拼图"玩法贯穿始终的设计。
- **橱窗网格：** 店铺中央有一个独立的、较小的展示柜（例如 4x4 或 5x5 的特殊网格）。
- **拼图连结：** 玩家对于高价值战利品，不只是简单收起来，而是尝试“拼”进橱窗。如果相邻的物品带有相同的标签（如带有【火】标签的剑与【火】标签的深渊矿石相邻），会触发“套装展览效应”，涉及物品的售价受到巨大的乘数加成。
- **VIP悬赏提交/展示：** 满足特定【VIP商会订单需求】的物品，可以直接在这里进行清算与高价交易。

### 3. 一键营业与蒙太奇结算动画（反馈与养成展现）
货摆好后，玩家点击**“开始营业 (Start Business)”**，随后切出 UI 界面，进入观赏期。
- **演出表现：** 播放一段快进的 Q版/横版小动画（蒙太奇）。时间快速流逝（UI上的月亮升起），魔偶站在柜台前，小人们（散客与土豪们）快速进进出出挑选商品。
- **视觉反馈：** 伴随每一位顾客离开，倾倒箱和橱窗里的对应物品快速消失，伴随清脆的金币声（+$150, +$800 等巨大数字在屏幕爆开），资产总额极速攀升。
- **生成账单：** 动画结束后弹出“今日经营账单”，汇总收益。

## 第二部分：资本循环引擎与行情法则（业务逻辑层）

这一切为了解决一个问题：不要让玩家对金钱失去概念，必须制造“经济通缩与焦虑”。

### 1. 账单倒排引擎 (Debt Deadline System)
驱动整个游戏重开或通关的唯一标尺。
- **全局时钟 `Time_Days`：** 每次核心抉择（深渊探险或停工休息）都会将时间推移一天。
- **固定月租：** 在特定的天数节点（如月末最后一天），必须缴纳一笔高昂的工坊租金和设备维护费。如果没有钱交，直接触发违约破产事件或强制典当。

### 2. 魔偶的经营属性体系
既然日常向路人兜售的工作通过抽象动画形式表现了，好感养成带来的数值就必须在此刻变现。
- **【算力】影响倾倒箱溢价：** 算力越高的魔偶，能把破铜烂铁说得天花乱坠，底价杂物的最终售价获得高百分比的利润加成。
- **【魅力】影响贵宾购买欲：** 魅力越高，吸引土豪客人的概率越高，能加速“镇店橱窗”极品物品的售出，甚至引发土豪直接无视底价清空橱窗。
- **性格标签的奇效：** 魔偶独有的性格特质会在结算动画里触发奇妙突发事件。
  - **[傲娇]：** 赶跑30%的普通看客，但剩下的被激起征服欲的客人会痛快支付双倍溢价。
  - **[腹黑]：** 有极低概率能把带有 `[负面诅咒]` 标签的危险遗物瞒天过海，以不可思议的天价卖给不明所以的路人。
  - **[迷糊]：** 偶尔算账出错导致当次交易小额亏损，但可能触发额外的长线补偿剧情（如次日好心人登门还钱）。

### 3. 价格波浪引擎 (Market Rumor System)
让“垃圾”在特定时机变成“宝藏”。
- **传闻生成：** 以周（Week）为单位，系统从库中抽取区域性经济事件卡（如：【深渊源石因战乱被高价抢购】）。
- **乘数生效：** 系统遍历配置表。如果在生效期间，玩家摆上橱窗的物品带有 `[OriginStone]` (源石) 标签，其基准单价无条件 `* 1.5`。
- **反馈至深渊（极其重要）：** 由于玩家看到了这个新闻。他在接下来下深渊时，看到原本极为占格子的“大型原始源石块”，会两眼放光把它硬塞进包里，甚至为此丢掉用于保命的回血药。这就是**经济指导战斗**的绝佳闭环保现。

### 4. 走私违禁品与隔夜税 (Contraband & Night-Tax)
深渊的好东西不能见光。
- **违禁标签：** 深渊产出的强力高阶战利品会带有 `[Contraband]` (违禁品) 标签。普通顾客根本不敢收，放进出货箱也会被退回。
- **黑市商人事件：** 只有利用特殊渠道传闻，或者在半夜触发特定的黑市走私商人上门剧情才能倾销。
- **隔夜税（仓储污染惩罚）：** 只要这种高污染货物未能出手，在魔偶所在的工坊里**过夜**（即时间推进了一天），就会向四周散发辐射，强制导致魔偶扣除不可逆的 `SAN（理智值）` 损伤。玩家每天都在“为了将来的巨款而雪藏”和“为了保住她的理智立刻扔掉”之间饱受精神折磨。

## 第三部分：原型样例库 (MVP Demos)

开发原型时所需的最简商业逻辑组件库。

### 1. 传闻事件卡配置示例
- **事件A：工会扫荡令**
  - **目标标签：** `Weapon`（武器类）
  - **行情修改：** `BasePrice * 1.3X`。因为城内戒严，黑市武器价格上涨。持续 3 天。
- **事件B：奇怪的迷信**
  - **目标标签：** `Slime`（史莱姆素材）
  - **行情修改：** `BasePrice * 0.2X`。镇上传闻史莱姆粘液会带来诅咒，价格暴跌。

### 2. 顾客 AI 预制体示例
- **【初级冒险者】**
  - **预算上限：** 300 G。
  - **偏好标签：** `Consumable` (药剂)。看到药剂时，愿意出高于底价 20% 的高价。
  - **砍价容忍：** 极低。如果你提价，他 80% 概率直接走人。
- **【蒙面买办】（概率出现）**
  - **预算上限：** 9999 G。
  - **偏好标签：** `Contraband` (违禁品)。只收常人不收的脏货，并能以远超原价的价格吃下。

## 第四部分：异常与边界情况 (Edge Cases)

| 极端情况 (Edge Case) | 系统裁决处理规范 (Resolution Rule) |
| :--- | :--- |
| **到了交租日，账面上的钱就差那 50 金币？** | **强制典当 (Forced Liquidation)：** 不会立刻 Game Over。系统会冻结你的操作权，强行打开你的安全仓库。你必须把包里留着当传家宝的图纸、准备用作核心构筑的暗金装备，以 **10%** 的跳楼废铁价扔进回收站，直到凑够所有的欠款。这种“割肉感”极其刺痛玩家。 |
| **我拒绝营业结算，把东西全屯在包裹里？** | **自动触发流转流逝：** 点击核心按钮结束一天时，必然无缝连接到营业环节。物品只要还在安全区域的存储系统内，即使当天未放入出货箱，也会随着时间面临耐久衰减及上文提到的违禁品辐射代价。 |
| **我今天选择了停工疗愈（没有下深渊），晚上还会营业吗？** | **照旧开店：** 每天晚上都一定会切入营业阶段动画。你可以用这段时间卖出之前几天下深渊囤积在仓库的存货。如果仓库空无一物强行营业，动画也会照常运转，只是不会有任何顾客进来，最终给玩家呈现一张残酷的 **$0 收益账单**，以此强化危机感。 |
| **我想把所有低价值普通物全放进橱窗蹭热度？** | **严格的空间惩罚：** VIP橱窗本身的空间网格极为狭小逼仄，你勉强塞进去的低底价物品虽然能获得微缩加成，但绝对损失依然巨大。而且如果橱窗未充分利用，还会白白浪费当日仅有一次的高昂热度资源。 |
