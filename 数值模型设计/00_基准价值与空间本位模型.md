# 基准价值与空间本位模型 (Space-Value Standard)

> 在“搜打撤+网格背包”游戏中，**金币不是唯一的一般等价物，【网格空间 (Grid Cell)】才是真正的底层本位币**。
> 所有的物品价值、战斗力、情感价值，最终都必须折算为它占据了多少个格子。

## 1. 空间本位基准 (The 1x1 Standard)

为了建立整个游戏的平衡，我们首先定义一个**【1x1 标准格】**在游戏中期的基准价值。

*   **定义：** 假设游戏处于“中期”（如第2个月，深渊第3层）。
*   **标准价值 (1 Cell Value, CV)：** `1 CV = 100 金币 (Gold)`。
*   这意味着，如果你带出一个纯粹卖钱的 1x1 战利品（无任何副作用），它的标准售价应该是 100 金币。

## 2. 物品价值折算公式

任何放入背包的物品，其价值构成可以分解为三个维度：**经济价值 (E)**、**战斗价值 (C)**、**情感/功能价值 (U)**。
一个平衡的物品，其总价值应该与其占据的网格数成正比。

**核心平衡公式：**
`网格数 (Grid Size) * CV = 经济价值(E) + 战斗价值(C) + 功能价值(U) + 负面状态补偿(D)`

### 2.1 战利品 (Loot) - 纯经济价值
*   **公式：** `售价 = 网格数 * CV * 稀有度系数 * 形状惩罚系数 + 负面状态补偿`
*   **形状惩罚：** 越难塞进背包的形状，单格价值越高。
    *   1x1 / 1x2 (长条): 系数 1.0
    *   2x2 (大方块): 系数 1.2 (因为它很难见缝插针)
    *   L型 / 异形: 系数 1.5
*   **负面状态补偿 (毒点)：** 如果带有“每回合掉1点SAN”的【毒性】标签，系统必须在原售价基础上给予额外补偿（比如单格价值提升50%），以奖励玩家承担风险。

### 2.2 武器/防具 (Gear) - 纯战斗价值
武器不直接产出金币，它占用空间是为了让你能在深渊活得更久，从而打出更多的格子来装战利品。
*   **1 CV 的战斗力等价：** `1 CV ≈ 10点 秒伤(DPS)` 或 `20点 等效生命值(EHP)`。
*   **推论：** 一把占用 1x3 (3格) 的长剑，其基础 DPS 期望应当在 `3 * 10 = 30 DPS` 左右。如果它带有特殊的连结 Buff，则基础 DPS 应当下调。

### 2.3 情感锚点物 (Anchor) - 纯功能价值
占用格子但不提供战斗力，纯粹为了稳定 SAN 值或触发奇迹。
*   **1 CV 的功能等价：** `1 CV ≈ 每场战斗结束恢复 5点 SAN` 或 `提供 2% 的免死概率`。

| 普通金块 | 1x1 | 100 金币 | 无 | 标准锚点 |

## 4. 动态价值波动（物价波浪）

基准价值 100G 并非固定不变。通过商店系统的 `GlobalPriceMultiplier`，我们可以控制整体经济的通胀与紧缩。
*   **平时：** 乘数 = 1.0 (标准价)
*   **清仓期：** 乘数 = 0.8 (逼迫玩家囤积高价值 1x1 物品)
*   **需求爆发期：** 特定标签（如【机械】）乘数 = 1.5 (鼓励玩家改变深渊下潜策略，专门去机械层进货)
