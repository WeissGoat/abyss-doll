# 战斗伤害与生存公式 (Combat & Survival Formulas)

> 本文档定义了局内“打”这个环节的数学模型。
> 背包的排列组合，最后都必须在这个系统里被验证为真实的生存能力。

## 1. 核心伤害公式 (Damage Formula)

伤害并不是简单的 `攻击 - 防御`，而是要考虑到物品在网格中的联动（Adjacency Buffs）和怪物的抗性。

**单次攻击实际伤害 (ActualDamage)：**
`ActualDamage = BaseDamage * (1 + AdjacencyMultiplier) * ElementModifier - MonsterArmor`

*   **BaseDamage：** 武器的基础伤害（例如：单手剑 15）。
*   **AdjacencyMultiplier：** 连结增幅。如果武器左侧放了一块【磨刀石 (+20%)】，下方放了一颗【充能核心 (+15%)】，则总连结系数为 35%。
*   **ElementModifier：** 属性克制系数（物理/魔法/精神）。通常为 1.0。若怪物带“重甲”且武器为物理，则为 0.5。
*   **MonsterArmor：** 怪物基础护甲。

**秒伤期望 (DPS)：**
这是衡量一把武器是否值得占用 N 个格子的唯一标准。
`DPS = ActualDamage / 武器充能时间 (Cooldown)`

*   **推论：** 一把高伤害长CD的武器（单次100，CD 10秒，DPS 10）和一把低伤害短CD的武器（单次10，CD 1秒，DPS 10），在占据同样格子数的情况下，前者更适合清爆发怪，后者适合持续输出。

## 2. 生存模型 (Survival Model - EHP)

人偶的生存不仅看血条，还要看防具能提供多少“等效生命值”。

**单局等效生命值 (Effective Health Pool, EHP)：**
`EHP = BaseHP + (ArmorBlocks * Mitigation)`

*   **BaseHP：** 基础耐久度（人偶自带）。
*   **ArmorBlocks：** 放在背包里的防具格子数。
*   **Mitigation：** 每个防具格提供的伤害减免。

**伤害结算流程：**
1.  怪物发动攻击（比如：20 伤害）。
2.  判定背包内是否有可充能的防具（如：木盾，每次抵挡 5 伤害，CD 3秒）。
3.  如果防具CD转好，则 `实际受到伤害 = 20 - 5 = 15`，扣除血量，防具进入CD。
4.  如果防具在CD中，则直接扣除 20 血量。

## 3. 理智侵蚀公式 (SAN Drain Math)

这是比血量更致命的隐性生存指标。
深渊中的怪物不仅打血，还会**“打SAN”**或者**“塞垃圾”**。

**当前理智值消耗池：**
`TotalSANCost = 移动税 + 战斗环境流失 + 诅咒物品流失 - 局内恢复`

*   **移动税 (Movement Tax)：** 每进入一个深渊新房间，强制扣除 2 点 SAN。这是逼迫玩家“不要贪图全图探索”的最强阀门。
*   **战斗环境流失：** 精英怪的特定技能（如“凝视”），不扣血，直接扣 5 点 SAN。
*   **诅咒物品流失 (Cursed Drain)：** 如果玩家贪婪地捡起了一个高价值战利品【眼球(带有毒性标签)】，只要它在背包里，每回合/每过一个房间就会额外扣除 1 点 SAN。
*   **局内恢复：** 背包里的【情感锚点物】（占用格子且无战斗力）能抵消掉落，或者吃极其昂贵的镇静剂（占用1x1，一次性+20SAN）。

## 4. 战斗极限压力的验证

在设计怪物数值时，策划需要验证以下条件：
*   **满血满防状态：** 能否顶着怪物的输出，通过武器的 DPS 强行换掉怪物。
*   **破产带毒状态：** 如果玩家为了钱，塞满了一包的高危战利品，没有武器和防具，怪物能否在 3 个回合内秒杀玩家（让其彻底失去一切）。
*   **SAN值归零惩罚：** 如果SAN归零，玩家不死亡，但【每受1点伤害，就永久销毁背包里的1件物品】。这将带来终极的恐惧体验。
