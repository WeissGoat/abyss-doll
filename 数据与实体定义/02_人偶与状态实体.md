# 人偶与状态实体定义 (Doll & Status Entities)

> 本文档定义了游戏中最核心的角色——人偶（Doll）的数据结构。人偶是承受深渊伤害、消耗资源以及建立情感连结的载体。

## 1. 人偶实体 (DollEntity)
人偶的根节点，通过组合多个组件来完整描述人偶的当前状态。

```json
{
  "entity": "DollEntity",
  "description": "人偶核心实体（基于组件化架构(CBA)的容器）",
  "properties": {
    "DollID": "string // 人偶配置表ID",
    "Name": "string // 人偶名字",
    "Status": "DollStatusComponent // 状态组件（血量/理智）",
    "Stats": "DollStatsComponent // 基础三维组件",
    "Chassis": "ChassisComponent // 底盘与网格组件",
    "Bond": "DollBondComponent // 好感度组件",
    "Traits": "List<string> // 当前附着在此人偶身上的特质/标签ID列表"
  }
}
```

## 2. 状态组件 (DollStatusComponent)
记录局内与局外波动的生存资源。**理智(SAN)和磨损(Wear)是这个品类的核心痛点。**

```json
{
  "entity": "DollStatusComponent",
  "description": "人偶生存状态组件",
  "properties": {
    "HP_Current": "int // 当前耐久度（归零局内战败）",
    "HP_Max": "int // 最大耐久度",
    "SAN_Current": "int // 当前理智值",
    "SAN_Max": "int // 最大理智值",
    "WearAndTear": "float // 磨损度 (0.0~1.0)，达到阈值会锁定网格或降低属性",
    "Corruption": "float // 侵蚀度 (0.0~1.0)，深渊毒属性累积，影响结算"
  }
}
```

## 3. 基础三维组件 (DollStatsComponent)
影响局外效率和局内数值换算的底层属性。

```json
{
  "entity": "DollStatsComponent",
  "description": "人偶基础属性",
  "properties": {
    "Power": "int // 机能：影响物理攻击折算、负重上限",
    "Compute": "int // 算力：影响能量攻击折算、营业卖货时的暴击率（高价溢出）",
    "Charm": "int // 魅力：影响好感度获取系数、势力交涉成功率"
  }
}
```

## 4. 底盘与网格组件 (ChassisComponent)
**这是搜打撤游戏的核心：决定了你有多大的“背包”。**

```json
{
  "entity": "ChassisComponent",
  "description": "机体底盘组件，定义初始网格形状",
  "properties": {
    "ChassisID": "string // 底盘配置ID",
    "Level": "int // 底盘进化阶位",
    "GridWidth": "int // 基础网格宽度 (X轴)",
    "GridHeight": "int // 基础网格高度 (Y轴)",
    "GridMask": "bool[][] // 初始网格可用性掩码（例如2x2的角落可能天生不可用，形成异形背包）",
    "LockedCells": "List<Vector2Int> // 被磨损度/诅咒物品强制锁死的格子坐标"
  }
}
```

## 5. 好感度组件 (DollBondComponent)

```json
{
  "entity": "DollBondComponent",
  "description": "情感双轨制数据",
  "properties": {
    "AffectionLevel": "int // 表面好感等级 (解锁语音、动作)",
    "HiddenTrust": "int // 隐藏信任值 (决定局内触发'奇迹锁血'的概率)"
  }
}
```

### 相关枚举与可选项 (Enums)

```cpp
enum DollState {
    Active = 0,      // 正常待命
    InDungeon = 1,   // 正在深渊中（锁定局外操作）
    Repairing = 2,   // 维护中（不可用）
    Broken = 3       // 大破（需要极高资源重置）
}
```
