# 经济与社会实体定义 (Economy & Social Entities)

> 本文档定义了小镇经营、势力订单和特质系统的数据。这里是深渊产出变现和形成经济压力的核心模块。

## 1. 商店与物价实体 (ShopInventory)
负责将垃圾战利品洗白为金币的机构。

```json
{
  "entity": "ShopInventory",
  "description": "动态物价与商品库存",
  "properties": {
    "CurrentWeek": "int // 当前物价周",
    "GlobalPriceMultiplier": "float // 全局物价波浪系数 (如 0.8 表示经济萧条)",
    "ItemTypeMultipliers": "Dictionary<ItemType, float> // 特定类型的物价浮动 (如本周 Weapon: 1.5倍)",
    "AvailableGoods": "List<ItemEntity> // 玩家可买入的消耗品/素材"
  }
}
```

## 2. 势力订单实体 (FactionOrder)
提供定向目标，占用极其宝贵的背包空间换取巨额回报。

```json
{
  "entity": "FactionOrder",
  "description": "组织/势力发布的委托订单",
  "properties": {
    "OrderID": "string // 订单ID",
    "FactionType": "FactionType // 发布势力",
    "RequiredItemConfigID": "string // 要求上交的特定物品",
    "DaysLeft": "int // 距离死线剩余天数",
    "RewardMoney": "int // 金钱回报",
    "RewardReputation": "int // 声望回报",
    "FailurePenalty": "int // 逾期未完成的声望惩罚"
  }
}
```

## 3. 标签与特质实体 (TraitEntity)
“你的行为塑造了她的性格”。记录深渊经历带来的长久影响。

```json
{
  "entity": "TraitEntity",
  "description": "附着在人偶或物品上的特质标签",
  "properties": {
    "TraitID": "string // 特质配置ID",
    "TraitType": "TraitType // 特质极性",
    "Name": "string // 如：'幽闭恐惧', '贪财'",
    "EffectLogicID": "string // 对应的逻辑脚本挂钩ID (供 ConditionEvaluator 读取)",
    "AccumulationValue": "int // 隐性累积值（例如被虫子打了10次才转化为特质）"
  }
}
```

## 4. 相关枚举与可选项 (Enums)

```cpp
enum FactionType {
    BlackMarket = 0, // 黑市（高风险高回报，扣除道德隐性值）
    Church = 1,      // 机械教会（提供高阶净化道具和图纸）
    Syndicate = 2    // 拾荒者辛迪加（提供基础生存物资打折）
}

enum TraitType {
    Positive = 1,    // 正面增益 (如：深渊抗性)
    Negative = -1,   // 负面惩罚 (如：恐虫症，遭遇虫子掉SAN加倍)
    Neutral = 0      // 中性改变 (如：狂热，攻击提升但无法后退)
}
```
