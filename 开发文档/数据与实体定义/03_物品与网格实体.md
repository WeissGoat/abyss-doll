# 物品与网格实体定义 (Item & Inventory Entities)

> 本文档定义了“背包管理”核心机制的基石：物品实体。
> 所有的武器、战利品、情感纪念物，在底层都是一个 `ItemEntity`，通过挂载不同的组件来实现不同的功能。

## 1. 物品核心实体 (ItemEntity)
任何能放进背包、仓库的东西，都是这个实体。

```json
{
  "entity": "ItemEntity",
  "description": "物品基础容器（组件化架构数据载体）",
  "properties": {
    "InstanceID": "string // 物品实例全局唯一ID（uuid）",
    "ConfigID": "string // 配置表ID（决定名字、图标等基础展示）",
    "ItemType": "ItemType // 物品大类枚举",
    "BaseValue": "int // 基础金币估值",
    "Grid": "ItemGridComponent // 网格形态组件（必带）",
    "Combat": "ItemCombatComponent // 战斗属性组件（武器/防具才有，战利品为空）",
    "Tags": "List<ItemTag> // 物品标签（如：毒、诅咒、活体、机械）"
  }
}
```

## 2. 网格形态组件 (ItemGridComponent)
**空间即资源。** 这个组件决定了物品长什么样，以及它占据多少“战力/资产”空间。

```json
{
  "entity": "ItemGridComponent",
  "description": "物品形状与占用组件",
  "properties": {
    "Shape": "Vector2Int[] // 形状坐标点集，例如1x2物品为 [(0,0), (0,1)]，L型为 [(0,0),(1,0),(0,1)]",
    "Rotation": "int // 当前旋转角度状态 (0, 90, 180, 270)",
    "GridCost": "int // 占用的总格数（计算体积比的依据，例如Shape长度为4，则为4）"
  }
}
```

## 3. 战斗与动作组件 (ItemCombatComponent)
当物品放入人偶背包并在深渊中触发时生效。战利品不挂载此组件。

```json
{
  "entity": "ItemCombatComponent",
  "description": "战斗行为组件",
  "properties": {
    "TriggerType": "TriggerType // 触发方式：手动点击、自动充能、受击触发等",
    "Cooldown": "float // 充能/冷却时间",
    "DamageType": "DamageType // 伤害/效果类型",
    "BaseValue": "int // 基础伤害/治疗/护盾数值",
    "AdjacencyBuffs": "List<AdjacencyEffect> // 相邻增幅规则（如：为右侧武器增加2点攻击）"
  }
}
```

## 4. 相关枚举与可选项 (Enums)

```cpp
enum ItemType {
    Weapon = 0,     // 武器（提供输出）
    Armor = 1,      // 防具（提供生存）
    Consumable = 2, // 消耗品（血药/净化剂，用完释放空间）
    Loot = 3,       // 战利品（纯卖钱，占用空间=负面Debuff）
    QuestItem = 4,  // 委托物（极大占用空间，不可丢弃）
    Anchor = 5      // 情感锚点物（纯爱礼物，提供局内Buff或SAN恢复，不提供直接战斗力）
}

enum ItemRarity {
    Common = 1,     // 白色（普通）
    Uncommon = 2,   // 蓝色（稀有）
    Rare = 3,       // 紫色（卓越）
    Epic = 4,       // 金色（史诗，通常带有复杂的网格连结特效）
    Cursed = 5      // 红色（诅咒，不可丢弃/带来持续理智掉落）
}

enum ItemTag {
    Biological = "活体", // 随时间可能变质
    Toxic = "毒性",      // 携带时持续掉SAN
    Heavy = "沉重",      // 降低人偶行动速度
    Fragile = "易碎"     // 受到攻击可能碎裂成低价值垃圾
}

enum TriggerType {
    OnCooldown = "冷却结束自动触发",
    OnHit = "击中敌人时触发",
    OnTakeDamage = "受到伤害时触发",
    Manual = "手动点击触发(消耗局内AP)"
}
```
